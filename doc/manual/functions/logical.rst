.. _function-logical:

#################
Logical Functions
#################

These functions provide scalar and SIMD version of logical operations.
Contrarily to scalar operations they **do not** provide short-circuit evaluations

The header ``#include <eve/function/logical.hpp>`` allows the inclusion of all specific logical
required headers in one shot.
