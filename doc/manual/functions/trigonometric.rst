########################
Trigonometric Functions
########################

These functions provide scalar and SIMD version of classical scalar trigonometric and inverse trigonometric functions.

The header ``#include <eve/function/trigonometric.hpp>`` allows the inclusion of all specific trigonometric functions required headers in one shot.

The header ``#include <eve/function/inverse_trigo.hpp>`` allows the inclusion of all specific inverse trigonometric functions required headers in one shot.
