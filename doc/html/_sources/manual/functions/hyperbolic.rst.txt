.. _function-hyperbolic:

########################
Hyperbolic Functions
########################

These functions provide scalar and SIMD version of classical scalar hyperbolic and inverse hyperbolic functions.

The header ``#include <eve/function/hyperbolic.hpp>`` allows the inclusion of all specific hyperbolic functions required headers in one shot.

The header ``#include <eve/function/inverse_hyperbolic.hpp>`` allows the inclusion of all specific inverse hyperbolic functions required headers in one shot.
