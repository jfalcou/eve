Mzero¶

Required header #include <eve/constant/mzero.hpp>

namespace eve
{
  constexpr as_arithmetic_t<Value> Mzero<Value>() noexcept
}

This ‘constant’ returns the Values -0, that is 0 for Integral types, but responding true to ref:is_negative <function-is_negative> for floating point types.

See also

is_negative, is_positive

Parameters¶

  • Value template type : the returned 0 is of type Value
