LogicalValue¶

A LogicalValue type is a type which refines Vectorizable or Vectorized while providing a boolean-like semantic.

Requirements¶

A type T satisfies LogicalValue if it refines Vectorizable or Vectorized and if :

  • eve::scalar_of_t<T> satisfies Logical

Example¶

  • eve::logical<double>
  • eve::wide<eve::logical<std::uint8_t>>
