Infix Operators¶

These functions provide scalar and SIMD version of the native C and C++ operators as functional interface.

Arithmetic Operators

For most of those operators, their semantic are the same than for regular language operator with the one restriction that both parameters must share a type. Note also that operations are performed within the input type’s range.

  • add
  • sub

Comparison Operators

Logical Operators

As usual for overloaded operators, beware that those operators lose their short-cut behavior when called on Vectorized types.

Bitwise Operators

Bitwise operations are extended to work on any floating-point types by considering the masking operations to apply on the value bit patterns while retaining its original types.

The header #include <eve/function/bitwise.hpp> allows the inclusion of all specific bitwise required headers in one shot.
