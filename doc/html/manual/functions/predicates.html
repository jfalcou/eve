Predicates FunctionsÂ¶

These functions provide scalar and SIMD version of logical predicates functions. Thes functions always return a Logical type For most of them, the semantic are the same as regular language operators or functions with the one restriction that both parameters must share a type unless specified otherwise and return a Logical value associated to this common type.
