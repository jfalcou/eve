modf¶

Required header: #include <eve/function/modf.hpp>

namespace eve
{
   constexpr /* implementation defined */ modf = {};
}

Function object Decomposing a Value into a fractional and integral part.

Synopsis¶

1
 template<typename T> constexpr std::pair<T, T> operator()( T x ) noexcept;
  • Computes a pair : a fractional and an integral part.

Parameter¶

  • x: Instance of a Value.

Return value¶

  • A pair of values with the same type as the parameter.

Notes¶

  • If the parameter is an Integral Value the fractional part is 0 ansd the integral part is x.

  • If the parameter is an Ieee Value:

    • If arg is \(\pm0\), {\(0\), \(\pm0\)} is returned.
    • If arg is NaN, {NaN, NaN} is returned.
    • If arg is \(\pm\infty\), {Nan, \(\pm\infty\)} is returned.
    • The call is equivalent to std::make_tuple(frac(x), trunc(x))

See also

frac, trunc

Example¶

#include <eve/function/modf.hpp>
#include <eve/wide.hpp>
#include <eve/constant/mindenormal.hpp>
#include <eve/constant/minf.hpp>
#include <eve/constant/inf.hpp>
#include <eve/constant/nan.hpp>

using wide_ft = eve::wide<float, eve::fixed<8>>;

int main()
{
  wide_ft pf = {-0.0, 
                1.30f,
                -1.3f,
                eve::Inf<float>(),
                eve::Minf<float>(),
                eve::Nan<float>(),
                0.678f,
                0.0f};

  auto [m, e] = eve::modf(pf); 
  std::cout << "---- simd" << '\n'
            << "<- pf =               " << pf << '\n'
            << "-> eve::modf(pf) =   [" << m << ", \n"
            << "                      " << e << "] \n"; 

  float xf = 2.3;
  auto [sm, se] =  eve::modf(xf); 

  std::cout << "---- scalar" << '\n'
            << "<- xf  =            " << xf << '\n'
            << "-> eve::modf(xf) = [" << sm << ", " << se << "]\n"; 
  return 0;
}

Possible output:

---- simd
<- pf =               (1.4013e-45, 1.3, -1.3, inf, -inf, -nan, 0.678, -2.7)
-> eve::modf(pf) =   [(1.4013e-45, 0.3, -0.3, -nan, -nan, -nan, 0.678, -0.7), 
                      (0, 1, -1, inf, -inf, -nan, 0, -2)] 
---- scalar
<- xf  =            2.3
-> eve::modf(xf) = [0.3, 2]
