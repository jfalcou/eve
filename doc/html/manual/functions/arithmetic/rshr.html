rshr¶

Required header: #include <eve/function/rshr.hpp>

namespace eve
{
   constexpr /* implementation defined */ rshr = {};
}

Function object performing a relative shift to the right or the left of an IntegralValue by another IntegralValue with the same number of elements and same element size. The shift direction is left (resp. right) if the shift is positive (resp. negative)

Synopsis¶

template<typename V, typename Shift> auto operator()( V const& x, Shift const& y ) noexcept;
  • Computes a bitwise right or left shift of each element of x by each element of y.

Parameters¶

  • Each parameter must be an instance of Value.
  • All Vectorized parameters must share the same cardinal
  • If the first parameter is Vectorizable, so must be the second

Return value¶

  • a value of the type of the first parameter.

Notes¶

  • shifts greater or equal to zero are to the right, less or equal to zero to the left.
  • shifts absolute values must not be greater or equal to the size in bit of the element shifted or an assert is issued.

See also

rshl, shl, shr, bit_shl, bit_shr

Example¶

#include <eve/function/rshr.hpp>
#include <eve/wide.hpp>

using iT      = std::int32_t;
using wide_it = eve::wide<iT, eve::fixed<4>>;

int main()
{
  wide_it pi = {100, 200, -2, 3};
  wide_it qi = {1, -2, 3, -1};

  std::cout << "---- simd" << '\n'
            << "<- pi =              " << pi << '\n'
            << "<- qi =              " << qi << '\n'
            << "-> eve::rshr(pi, qi) = " << eve::rshr(pi, qi) << '\n';

  iT xi = 2, mxi = -2, yi = 3;

  std::cout << "---- scalar" << '\n'
            << "<- xi  =                      " << xi << '\n'
            << "<- mxi =                      " << mxi << '\n'
            << "<- yi  =                      " << yi << '\n'
            << "-> eve::rshr(xi, yi) = " << eve::rshr(xi, yi) << '\n'
            << "-> eve::rshr(mxi, yi)= " << eve::rshr(mxi, yi) << '\n';
  return 0;
}

Possible output:

---- simd
<- pi =              (100, 200, -2, 3)
<- qi =              (1, -2, 3, -1)
-> eve::rshr(pi, qi) = (50, 800, -1, 6)
---- scalar
<- xi  =                      2
<- mxi =                      -2
<- yi  =                      3
-> eve::rshr(xi, yi) = 0
-> eve::rshr(mxi, yi)= -1
