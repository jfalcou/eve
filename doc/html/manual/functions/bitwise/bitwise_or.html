bitwise_or¶

Required header: #include <eve/function/bitwise_or.hpp>

namespace eve
{
   constexpr /* implementation defined */ bitwise_or = {};
}

Function object performing a bitwise OR between two Values of same size.

Synopsis¶

1
2
3
 template<typename T, typename M, typename U, typename N> wide<T,N> operator()( wide<T,N> const& v, wide<U,M> const& w ) noexcept;
 template<typename T, typename N, typename U>             wide<T,N> operator()( wide<T,N> const& v, U s ) noexcept;
 template<typename T, typename U> constexpr               T         operator()( T s, U t ) noexcept;
  • [1] Performs a bitwise OR between both wides <type-wide>.
  • [2] Performs a bitwise OR between the scalar and each elements of the wide instance.
  • [3] Performs a bitwise OR between both scalars.

Parameters

  • v, w: Instances of wide satisfying sizeof(v) == sizeof(w).
  • s, t: Scalar values of type U satisfying sizeof(T) == sizeof(U).

Return value

  • [1,2] A value with the same type as the first parameter.
  • [3] A value of type T.

Notes

  • There is no type restriction between operands of function-bitwise_or as long as the number of bits between them are equals. This implies that calls to:ref:function-bitwise_or on floating points values are possible as long as they are performed with a second parameters of proper size.
  • There is no cardinal restriction on the Vectorized operands of function-bitwise_or as long as the number of bits between them are equals. This implies that calls to:ref:function-bitwise_or on Vectorized values of different cardinals are allowed as long at their total size in bits are equal.

Example¶

#include <eve/function/bitwise.hpp>
#include <eve/constant/nan.hpp>
#include <eve/wide.hpp>
#include <iostream>

using wide_it = eve::wide<std::uint32_t, eve::fixed<4>>;

int main() {
  wide_it pi = {3, 6, 5, 27};
  wide_it qi = {4, -1, 2, ~0};
  std::cout << "---- simd" << '\n'
            << " <- pi =                     " << pi << '\n'
            << " <- qi =                     " << qi << '\n'
            << " -> eve::bitwise_or(pi, qi) = " << eve::bitwise_or(pi, qi)
            << '\n';

  std::uint32_t xi = 3, yi = 4;

  std::cout << "---- scalar" << '\n'
            << " xi =                        " << xi << '\n'
            << " yi =                        " << yi << '\n'
            << " -> eve::bitwise_or(xi, yi) = " << eve::bitwise_or(xi, yi)
            << '\n';
  return 0;
}

Possible output:

---- simd
 <- pi =                     (3, 6, 5, 27)
 <- qi =                     (4, 4294967295, 2, 4294967295)
 -> eve::bitwise_or(pi, qi) = (7, 4294967295, 7, 4294967295)
---- scalar
 xi =                        3
 yi =                        4
 -> eve::bitwise_or(xi, yi) = 7
