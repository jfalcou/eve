bitwise_not¶

Required header: #include <eve/function/bitwise_not.hpp>

namespace eve
{
   constexpr /* implementation defined */ bitwise_not = {};
}

Function object performing a bitwise COMPLEMENT of a Value.

Synopsis¶

1
2
 template<typename T, typename M, typename N> wide<T,N> operator()( wide<T,N> const& v ) noexcept;
 template<typename T, typename U> constexpr   T         operator()( T s ) noexcept;
  • [1] Performs the bitwise COMPLEMENT of v.
  • [2] Performs the bitwise COMPLEMENT of s.

Parameters

  • v, w: Instance of wide .
  • s, t: Scalar value.

Return value

  • [1,2,3] A value with the same type as the parameter.

Notes

  • There is no type restriction on the parameter Value type of bitwise_not. This implies that calls to:ref:function-bitwise_not on IEEEvalues are possible.

Example¶

#include <eve/function/bitwise.hpp>
#include <eve/wide.hpp>
#include <iostream>

using wide_it = eve::wide<std::uint32_t, eve::fixed<4>>;

int main() {
  wide_it pi = {1, 2, ~0, 1023};

  std::cout << "---- simd" << '\n'
            << "<- pi =                  " << pi << '\n'
            << "-> eve::bitwise_not(pi) = " << eve::bitwise_not(pi) << '\n';

  std::uint32_t xi = 2;

  std::cout << "---- scalar" << '\n'
            << "<- xi =                  " << xi << '\n'
            << "-> eve::bitwise_not(xi) = " << eve::bitwise_not(xi) << '\n';
  return 0;
}

Possible output:

<- pi =                  (1, 2, 4294967295, 1023)
-> eve::bitwise_not(pi) = (4294967294, 4294967293, 0, 4294966272)
---- scalar
<- xi =                  2
-> eve::bitwise_not(xi) = 4294967293
