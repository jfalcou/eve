sub¶

Required header #include <eve/function/sub.hpp>

namespace eve
{
  constexpr Value sub( Value lhs, Value rhs ) noexcept
}

Function object for performing substarction over two Value of the same type.

Parameters¶

  • lhs, rhs : values to substract

Return value¶

The sum of arg0 and arg1 for every element of each parameters.

Options¶

Example¶

 #include <eve/function/sub.hpp>
 #include <eve/wide.hpp>

 int main()
 {
    using namespace eve::literal;
    eve::wide<float> w( [](int i, int) { return 1.f+i; } );

    std::cout << "sub(3.f, 5.5f) = " << eve::sub(3.f,5.5f) << '\n';
    std::cout << "sub({3,3,3,3}, {1,2,3,4.5}) = " << eve::sub(3_wf, w ) << '\n';
 }

Possible output

sub(3.f, 5.5f) = -2.5
sub(3_wf, {1,2,3,4.5}) = {2,1,0,-1.5}
