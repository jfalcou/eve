add¶

Required header #include <eve/function/add.hpp>

namespace eve
{
  constexpr Value add( Value lhs, Value rhs ) noexcept
}

Function object for performing addition over two Value of the same type.

Parameters¶

  • lhs, rhs : values to sum

Return value¶

The sum of arg0 and arg1 for every element of each parameters.

Options¶

Example¶

#include <eve/function/add.hpp>
#include <eve/literals.hpp>
#include <eve/wide.hpp>

int main()
{
  using namespace eve::literal;
  eve::wide<float> w( [](int i, int) { return 1.f+i; } );

  std::cout << "add(3.f, 5.5f) = " << eve::add(3.f,5.5f) << '\n';
  std::cout << "add({3,3,3,3}, {1,2,3,4}) = " << eve::add(3._wf, w ) << '\n';
}

Possible output

add(3.f, 5.5f) = 8.5
add(3_wf, {1,2,3,4}) = {4,5,6,7}
