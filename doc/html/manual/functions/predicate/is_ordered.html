is_ordered¶

Required header: #include <eve/function/is_ordered.hpp>

namespace eve
{
   constexpr /* implementation defined */ is_ordered = {};
}

Function object performing ‘none is NaN’ test with two Values.

Synopsis¶

1
2
3
4
 template<typename T, typename N>             as_logical_t<wide<T,N>> operator()( wide<T,N> const& v, wide<T,N> const& w ) noexcept;
 template<typename T, typename N, typename U> as_logical_t<wide<T,N>> operator()( wide<T,N> const& v, U s ) noexcept;
 template<typename T, typename N, typename U> as_logical_t<wide<T,N>> operator()( U s, wide<T,N> const& v ) noexcept;
 template<typename T, typename U> constexpr   as_logical_t<T>         operator()( T s, U t ) noexcept;
  • [1] Performs element-wise ‘none is NaN’ test between both wides.
  • [2,3] Performs ‘none is NaN’ test between the scalar and each element of the wide instance.
  • [4] Performs ‘none is NaN’ test between both scalars.

Parameters

  • v, w: Instances of wide.
  • s, t: Scalar values of type U which must be convertible to T.

Return value

  • [1,2] A value of type as_logical_t<wide<T,N>>.
  • [3] A value of type as_logical_t<T>.

Notes

  • is_ordered(x,y) is equivalent to is_not_NaN(x) && is_not_NaN(y)
  • The return type is not bool as predicates applied on Vectorized types need to store multiple values.

Example¶

#include <eve/function/predicates.hpp>
#include <eve/wide.hpp>
#include <eve/constant/inf.hpp>
#include <eve/constant/minf.hpp>
#include <eve/constant/nan.hpp>
#include <eve/constant/mindenormal.hpp>
#include <iostream>

using wide_ft = eve::wide<float, eve::fixed<8>>;

int main()
{
  wide_ft pf = {0.0f,
                1.0f,
                -1.0f,
                -2.0f,
                eve::Mindenormal<float>(),
                eve::Inf<float>(),
                eve::Minf<float>(),
                eve::Nan<float>()};
  wide_ft qf = {eve::Mindenormal<float>(),
                eve::Inf<float>(),
                eve::Minf<float>(),
                eve::Nan<float>(),
                0.0f,
                1.0f,
                -1.0f,
                -2.0f};

  std::cout << "---- simd" << '\n'
            << "<- pf =                  " << pf << '\n'
            << "<- qf =                  " << qf << '\n'
            << "-> eve::is_ordered(pf, qf) = " << eve::is_ordered(pf, qf) << '\n';

  float xf = 1.0f;
  float yf = eve::Nan<float>();

  std::cout << "---- scalar" << '\n'
            << "<- xf =                  " << xf << '\n'
            << "<- yf =                  " << yf << '\n'
            << "-> eve::is_ordered(xf, yf) = " << eve::is_ordered(xf, yf) << '\n';
  return 0;
}

Possible output:

---- simd
<- pf =                  (0, 1, -1, -2, 1.4013e-45, inf, -inf, -nan)
<- qf =                  (1.4013e-45, inf, -inf, -nan, 0, 1, -1, -2)
-> eve::is_ordered(pf, qf) = (true, true, true, false, true, true, true, false)
---- scalar
<- xf =                  1
<- yf =                  -nan
-> eve::is_ordered(xf, yf) = false
