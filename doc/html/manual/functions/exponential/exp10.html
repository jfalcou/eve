exp10¶

Required header: #include <eve/function/exp10.hpp>

namespace eve
{
   constexp10r /* implementation defined */ exp10 = {};
}

Function object computing the base 10 exponential of(\(10^x\)) of an IEEEValue.

Synopsis¶

1
2
 template<typename T, typename N>  wide<T,N> operator()( wide<T,N> const& v) noexcept;
 template<typename T> constexp10r    T         operator()( T s ) noexcept;
  • [1] Computes the element-wise base 10 exponential of the wide.
  • [2] Computes the base 10 exponential of the scalar.

Parameters

  • v: Instance of IEEEValue.
  • s: Scalar value.

Return value

  • [1,2] A value with the same type as the parameter.

See also

exp, exp10, expm1, log, log, log10, log1p,

Example¶

#include <eve/function/exp10.hpp>
#include <eve/wide.hpp>
#include <eve/constant/inf.hpp>
#include <eve/constant/minf.hpp>
#include <eve/constant/nan.hpp>
#include <eve/constant/mindenormal.hpp>
#include <iostream>

using wide_ft = eve::wide<float, eve::fixed<8>>;

int main()
{
  wide_ft pf = {0.0f,
                1.0f,
                -1.0f,
                -2.0f,
                eve::Mindenormal<float>(),
                eve::Inf<float>(),
                eve::Minf<float>(),
                eve::Nan<float>()};

  std::cout << "---- simd" << '\n'
            << "<- pf =                      " << pf << '\n'
            << "-> eve::exp10(pf) =            " << eve::exp10(pf) << '\n'; 

  float xf = 1.0f;
  float yf = eve::Nan<float>();

  std::cout << "---- scalar" << '\n'
            << "<- xf =                      " << xf << '\n'
            << "-> eve::exp10(xf) =            " << eve::exp10(xf) << '\n'
            << "<- yf =                      " << yf << '\n'
            << "-> eve::exp10(yf) =            " << eve::exp10(yf) << '\n'; 
  return 0;
}

Possible output:

---- simd
<- pf =                      (0, 1, -1, -2, 1.4013e-45, inf, -inf, -nan)
-> eve::exp10(pf) =            (1, 10, 0.1, 0.01, 1, inf, 0, -nan)
---- scalar
<- xf =                      1
-> eve::exp10(xf) =            10
<- yf =                      -nan
-> eve::exp10(yf) =            nan
