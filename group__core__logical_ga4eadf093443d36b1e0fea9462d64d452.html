Parameters

  • x, y: arguments.

Return value

Returns the logical OR of the two parameters following the logical operations semantic.

The call logical_or(x, y) is semantically equivalent to x || y if x or y is an simd value and does not shortcut.

Note
Although the infix notation with || is supported, the || operator on standard scalar types is the original one and so will return bool instead of eve::logical_value.

Example

#include <eve/module/core.hpp>
#include <eve/wide.hpp>
#include <iostream>
using wide_ft = eve::wide<float, eve::fixed<4>>;
int main()
{
using eve::is_even;
using eve::is_odd;
wide_ft pf = {3.0f, -2.0f, -3.0f, 51.04f};
wide_ft qf = {4.0f, -1.0f, -3.0f, 0.0f};
std::cout << "---- simd" << '\n'
<< " <- pf = " << pf << '\n'
<< " <- qf = " << qf << '\n'
<< " -> eve::logical_or(is_odd(pf), is_even(qf)) = " << eve::logical_or(is_odd(pf), is_even(qf)) << '\n';
float xf = 3.0f, yf = 4.5f;
std::cout << "---- scalar" << '\n'
<< " xf = " << xf << '\n'
<< " yf = " << yf << '\n'
<< " -> eve::logical_or(is_odd(xf), is_even(yf)) = " << eve::logical_or(is_odd(xf), is_even(yf)) << '\n'
<< " -> eve::logical_or(xf == 3, is_even(yf)) = " << eve::logical_or(xf == 3 , is_even(yf)) << '\n' ;
return 0;
}
