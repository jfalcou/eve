template<typename C>
struct eve::if_< C >

Extensible wrapper for SIMD conditional.

Required header: #include <eve/conditional.hpp>

eve::if_ wraps a eve::logical value so that it can be either extended with an alternative value or help optimize conditional operations evaluation.
