◆ cyl_bessel_in

eve::cyl_bessel_in = {}
inlineconstexpr

Callable object computing the cyl_bessel_in function, \( I_{n}(x)=\left(\frac12z\right)^n\sum_{k=0}^{\infty}{\frac{(x^2/4)^k}{k!\,\Gamma (k+n +1)}}\).

Required header: #include <eve/module/bessel.hpp>

Members Functions

Member Effect
operator() the cyl_bessel_in function
template< real_value N, floating_real_value T > auto operator()( N n, T x ) const noexcept;

Parameters

n: real_value order of the function (non necessarily integral)

x: positive floating_real_value.

Return value

\(\displaystyle I_{n}(x)=\left(\frac12z\right)^n\sum_{k=0}^{\infty}{\frac{(x^2/4)^k}{k!\,\Gamma (k+n +1)}}\).

It is the solution of \( x^{2}y''+xy'+(x^2-n^2)y=0\) for which \( y(0) = 0\).

Supported decorators

  • eve::diff, eve::diff_1st, eve::diff_nth
 The expression `eve::diff(eve::cyl_bessel_in)(n, x)` computes the derivative of the function at `x`.

Example

See it live on Compiler Explorer

#include <eve/module/bessel.hpp>
#include <eve/wide.hpp>
#include <iostream>
using wide_ft = eve::wide<double, eve::fixed<8>>;
int main()
{
wide_ft x = {0.5, 1.5, 0.1, 1.0, 19.0, 25.0, 21.5, 10000.0};
wide_ft n = {0.5, -1.0, 1.5, -2.0, 2.5, -2.6, 3.2, -12};
std::cout << "---- simd" << '\n'
<< "<- n = " << n << '\n'
<< "<- x = " << x << '\n'
<< "-> cyl_bessel_in(n, x) = " << eve::cyl_bessel_in(n, x) << '\n'
<< "-> diff(cyl_bessel_in)(n, x) = " << eve::diff(eve::cyl_bessel_in)(n, x) << '\n';
double xd = 1.0;
std::cout << "---- scalar" << '\n'
<< "<- xd = " << xd << '\n'
<< "-> cyl_bessel_in(3, xd) = " << eve::cyl_bessel_in(3, xd) << '\n';
return 0;
}
