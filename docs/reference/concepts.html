**Expressive Vector Engine** (insert ../crumbs.html here) # Concepts The following concepts provide definitions of fundamental library concepts used throughout **EVE** to perform compile-time validation of template arguments and perform function dispatch based on properties of those types. Based on the general properties and reasoning style from the C++ Standard library, **EVE** concepts caters to the specific needs of SIMD computations. **Convenience header:** ## Base Concepts [`eve::rebindable`](concepts/rebindable.html) : Type supporting structured binding [`eve::scalar_value`](concepts/scalar_value.html) : Type supporting vectorization [`eve::simd_value`](concepts/simd_value.html) : Type supporting vectorized operations [`eve::value`](concepts/value.html) : Type usable as **EVE** callable objects' parameters [`eve::element_compatible_to`
`eve::compatible_values`](concepts/compatible.html) : Verify the constraints between scalar and SIMD types for arithmetic-like operations. [`eve::element_bit_compatible_to`
`eve::bit_compatible_values`](concepts/bit_compatible.html) : Verify the constraints between scalar and SIMD types for bitwise-like operations. ## Convenience Concepts The following concepts provide short-cuts to recurrent combination of **EVE** concepts and some properties of its element's type. ### Scalar-based Convenience Concepts **Required header:** **`eve::integral_scalar_value`** : Integral type that satisfies [**`eve::scalar_value`**](concepts/scalar_value.html) `eve::signed_scalar_value` : Signed type that satisfies [**`eve::scalar_value`**](concepts/scalar_value.html) `eve::unsigned_scalar_value` : Unsigned type that satisfies [**`eve::scalar_value`**](concepts/scalar_value.html) `eve::signed_integral_scalar_value` : Signed integral type that satisfies [**`eve::scalar_value`**](concepts/scalar_value.html) `eve::floating_scalar_value` : Floating point type that satisfies [**`eve::scalar_value`**](concepts/scalar_value.html) `eve::logical_scalar_value` : Logical type that satisfies [**`eve::scalar_value`**](concepts/scalar_value.html) ### SIMD-based Convenience Concepts **Required header:** **`eve::integral_simd_value`**
