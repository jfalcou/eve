**Expressive Vector Engine** IEEE Constants (insert ../../crumbs.html here) Those constants provides generator function for constants tied to the IEEE standard or that ease implementation of IEEE standard functions. Synopsis ==================================================================================================== All constant are implemented as callable object taking a [type specifier](../types/helpers.html#eve::as_) as argument. Each constant is available by including the header file `eve/constant/name.hpp`, where `name` is the constant name. **Example:** **Output:** List of constants
