**Expressive Vector Engine** Numeric Limits (insert ../../crumbs.html here) Those constants provides generator function for limiting values constants. Synopsis ==================================================================================================== All constant are implemented as callable object taking a [type specifier](../types/helpers.html#eve::as_) as argument. Each constant is available by including the header file `eve/constant/name.hpp`, where `name` is the constant name. **Example:** **Output:** List of constants ==================================================================================================== $$\newcommand{\choose}[4]{\left\{ \begin{array}{@{}ll@{}} #1 & \texttt{if E is } \texttt{#2}\ \\ #3 & \texttt{if E is } \texttt{#4} \end{array}\right.}$$ $$\newcommand{\choosefd}[2]{\left\{ \begin{array}{@{}ll@{}} #1 & \texttt{if E is float } \\ #2 & \texttt{if E is double} \end{array}\right.}$$ $$\newcommand{\tchoosefd}[2]{ \choosefd{\texttt{#1}}{\texttt{#2}}}$$ $$\newcommand{\choosefdi}[3]{\left\{ \begin{array}{@{}ll@{}} #1 & \texttt{if E is float } \\ #2 & \texttt{if E is double} \\ #3 & \texttt{if E is integral} \end{array}\right.}$$ In the following table: * `T` represents the requested type for the generated constant. * `E` represents the [element type](../../glossary.html#elementtype#elementtype) associated to `T` Description | Callable object | Value | Constraints on `T` | :-----------------------------------------------|:-----------------------|:-----------------|:-------------------| Infinity | `eve::inf` | $$\infty$$ | `floating_value` | Minus infinity | `eve::minf` | $$-\infty$$ | `floating_value` | Greatest value | `eve::valmax` | $$\tchoosefd{3.40282e+38f}{1.79769313486232e+308}$$ | `value` | Least value | `eve::valmin` | $$\tchoosefd{-3.40282e+38f}{-1.79769313486232e+308}$$ | `value` | Greatest contiguous integral floating value | `eve::maxflint` | $$\tchoosefd{16777216.0f}{9007199254740992.0}$$ | `floating_value` | Smallest strictly positive normal value | `eve::smallestposval` | $$\tchoosefd{1.17549e-38f}{2.2250738585072e-308}$$ | `floating_value` | Square root of greatest value | `eve::sqrtvalmax` | $$\tchoosefd{1.844674e+19f}{1.340780792994260e+154}$$ | `floating_value` | ----------------------------------------------------------------------------------------------------
