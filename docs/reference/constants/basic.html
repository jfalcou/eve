**Expressive Vector Engine** Basic Constants (insert ../../crumbs.html here) Those constants provides generator function for basic values used in various settings. Synopsis ==================================================================================================== All constant are implemented as callable object taking a [type specifier](../types/helpers.html#eve::as_) as argument. Each constant is available by including the header file `eve/constant/name.hpp`, where `name` is the constant name. **Example:** **Output:** List of constants ==================================================================================================== In the following table: * `T` represents the requested type for the generated constant. * `E` represents the [element type](../../glossary.html#elementtype#elementtype) associated to `T` Constant | Callable object | Value | Constraints on `T` | :-------------|:----------------|:--------------------------|:---------------------| All bits set | `eve::allbits` | see below | `arithmetic_value` | True | `eve::true` | `as_logical_t(true)` | `value` | False | `eve::false` | `as_logical_t(false)` | `value` | Half | `eve::half` | `T(0.5)` | `floating_value` | Minus half | `eve::mhalf` | `T(-0.5)` | `floating_value` | One | `eve::one` | `T(1)` | `arithmetic_value` | Minus one | `eve::mone` | `T(-1)` | `arithmetic_value` | Zero | `eve::zero` | `T(0)` | `arithmetic_value` | Minus Zero | `eve::mzero` | `T(-0)` | `arithmetic_value` | ----------------------------------------------------------------------------------------------------
