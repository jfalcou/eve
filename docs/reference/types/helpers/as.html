eve::as_ ==================================================================================================== **Required header:** **Synopsis:** Type wrapping structure. Various **EVE** functions requires to be configured by a type. This type argument is used for selecting or configuring the output type of the functions (i.e the various [conversions] functions). Instead of requiring users to pass unwieldy template parameters that may clash with the [function objects paradigm](../rationale.html#functionobjectsasapi) **EVE** uses, Instances of [`eve::as`](#as) can be passed as objects and the type they wrap can be retrieved later via their member type. **Member types:** Member type | Definition ------------ | ---------- `type` | `T` **Example:** eve::as ==================================================================================================== **Required header:** **Synopsis:** Return an instance of [`eve::as_`](#as) wrapping the type of `x`. **Parameters:** `x` : value which type will be wrapped. **[optional]** **Return Value:** An instance of [`eve::as_`](#as)`>`. **Example:** Predefined Type Wrappers ==================================================================================================== **Required header:** **Synopsis:** 1.
