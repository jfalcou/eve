eve::as_wide ==================================================================================================== **Required header:** Build a [SIMD type](../../../concept.html#simd_value) from a given [element type](traits.html#element_type) `Type` and an optional [cardinal](traits.html#cardinal) `Size`. - If `Type` satisfies [scalar_value](../../../concept.html#scalar_value), `eve::as_wide<Type,Cardinal>::type` evaluates to `wide<Type,Cardinal>`. - If `Type` satisfies [simd_value](../../../concept.html#simd_value), `eve::as_wide<Type,Cardinal>::type` evaluates to `wide<eve::element_t<Type>,Cardinal>`, i.e it overwrites the initial type's cardinal. - If `Type` is a template type satisfies Rebindable, `eve::as_wide<Type,Cardinal>::type` evaluates to `std::tuple<eve::as_wide_t<Ts>...>`. - If `Type` is an instance of `std::pair` or `std::array`, evaluates and instance of the same type but with its parameters replaced by the proper evaluation of `eve::as_wide_t`. **Member types:** Member type | Definition ------------ | ---------- `type` | The [SIMD type](../../../concept.html#simd_value) build from `Type` and `Cardinal` **Helper Types:** **Example:** **Possible Output:** ----------------------------------------------------------------------------------------------------
