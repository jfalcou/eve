**Expressive Vector Engine** `eve::ceil`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the ceil operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) ceil operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) the smallest integer not less than `x`. Supported Decorators ==================================================================================================== [`tolerant`](../decorators.html#tolerant) : The expression `tolerant(ceil)(x, tol)` computes a tolerant ceil value for `x`, where `x` must be a floating value. - If `tol` is a floating_value computes the floor with a tolerance `tol` using Hagerty's FL5 function. - If `tol` is an integral_value n compute the floor of the next nth representable value in the `x` type. - If `tol` is omitted the tolerance is taken to 3 times the machine $\epsilon$ in the `x` type (`3*eps(as(x))`).
