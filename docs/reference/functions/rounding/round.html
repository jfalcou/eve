**Expressive Vector Engine** `eve::round`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the round operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) round operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) the integer nearest to `x`. If `x` is an exact half-integer the rouding is made to the nearest even integer. The call to `round(a)` is equivalent to the call `nearest(a)`. Supported decorators ==================================================================================================== [`downward`](../decorators.html#downward) : The expression `downward(round)(x)` is equivalent to `floor(x)`. [`upward`](../decorators.html#upward) : The expression `upward(round)(x)` is equivalent to `ceil(x)`. [`to_nearest`](../decorators.html#to_nearest) : The expression `to_nearest(round)(x)` is equivalent to `nearest(x)`. [`toward_zero`](../decorators.html#toward_zero) : The expression `toward_zero(round)(x)` is equivalent to `trunc(x)`. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(round)(x)` computes the derivative of the function at `x` (it is the same as `nearest`).
