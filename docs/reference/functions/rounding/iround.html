**Expressive Vector Engine** `eve::iround`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the iround operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) iround operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) the integer nearest to `x`. If `x` is an exact half-integer the rouding is made to the nearest even integer. The call to `iround(a)` is equivalent to the call `inearest(a)`. !!! WARNING If the result is not representable in the return type, it is undefined. Supported Decorators ==================================================================================================== [`downward`](../decorators.html#downward) : The expression `downward(iround)(x)` is equivalent to `ifloor(x)`. [`upward`](../decorators.html#upward) : The expression `upward(iround)(x)` is equivalent to `iceil(x)`. [`to_nearest`](../decorators.html#to_nearest) : The expression `to_nearest(iround)(x)` is equivalent to `inearest(x)`. [`toward_zero`](../decorators.html#toward_zero) : The expression `toward_zero(iround)(x)` is equivalent to `itrunc(x)`.
