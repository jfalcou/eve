**Expressive Vector Engine** `eve::lbeta`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the lbeta operation. Member functions ==================================================================================================== Performs the computation of $\displaystyle the logarithm of \mbox{B}(x,y)=\int_0^1 t^{x-1}(1-t)^{y-1}\mbox{d}t$. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) $\displaystyle \log[\frac{\Gamma(x)\Gamma(y)}{\Gamma(x+y)}}$. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative_1st(lbeta)(x,y)` and `derivative_2nd(lbeta)(x,y)` computes the partial derivatives of $f$, where $f$ is the function $(x,y) \rightarrow \ \log(\mbox{B}(x,y))$.
