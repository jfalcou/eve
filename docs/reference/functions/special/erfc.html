**Expressive Vector Engine** `eve::erfc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the erfc operation. Member functions ==================================================================================================== Performs the computation of the complementary error function. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) $\displaystyle \mbox{erfc}(x)=\frac{2}{\sqrt\pi}\int_{x}^{\infty} e^{-t^2}\mbox{d}t$ The result type is of the same type as the parameter. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(erfc)(x)` computes the derivative of the function at `x`.
