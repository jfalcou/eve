**Expressive Vector Engine** `eve::erfcx`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the erfcx operation. Member functions ==================================================================================================== Performs the computation of the scaled complementary error function. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) $\displaystyle \mbox{erfcx}(x)=\frac{2e^{x^2}}{\sqrt\pi}\int_{x}^{\infty} e^{-t^2}\mbox{d}t$ The result type is of the same type as the parameter. Supported decorators ==================================================================================================== [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(erfcx)(x)` computes the derivative of the function at `x`.
