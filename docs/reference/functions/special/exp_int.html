Member functions ==================================================================================================== Performs the computation of $\displaystyle E_{n}(x)=\int_0^\infty \frac{e^{-xt}}{t^{n}} \mbox{d}t$$ Parameters ---------------------------------------------------------------------------------------------------- `n` : [value](../../concepts.html#value). `x` : [floating value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) exponential integral The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. If the first parameter inot an integral value it must be a flint, otherwise the result is Nan. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(exp_int)(n,x)` computes the derivative (relative to the second parameter) of the function at `x`.
