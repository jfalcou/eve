**Expressive Vector Engine** `eve::betainc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the betainc operation. Member functions ==================================================================================================== Performs the computation of $\displaystyle \mbox{B}(s,x,y)=\int_0^s t^{x-1}(1-t)^{y-1}\mbox{d}t$, or equivalently $\displaystyle \frac{\Gamma(x)\Gamma(y)}{\Gamma(x+y)}$ Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the value of the incomplete beta function. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative_1st(betainc)(s,x,y)`, `derivative_2nd(betainc)(s,x,y)` and `derivative_3rd(betainc)(s,x,y)` computes the partial derivatives of $f$, where $f$ is the function $(s,x,y) \rightarrow \ \mbox{B}(s,x,y)$.
