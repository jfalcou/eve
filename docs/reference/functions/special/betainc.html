**Expressive Vector Engine** `eve::betainc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the betainc operation. Member functions ==================================================================================================== Performs the computation of $\displaystyle \mbox{B}(s,x,y)=\frac{1}{\mbox{B}(x,y)}\int_0^s t^{x-1}(1-t)^{y-1}\mbox{d}t$ Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the value of the incomplete beta function. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(betainc)(s,x,y)`, computes the derivative of $f_{x,y}$, where $f_{x,y}$ is the function $s \rightarrow \ \mbox{B}(s,x,y)$. The other partial derivative are still unimplemented.
