**Expressive Vector Engine** `eve::beta`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the beta operation. Member functions ==================================================================================================== Performs the computation of $\displaystyle \mbox{B}(x,y)=\int_0^1 t^{x-1}(1-t)^{y-1}\mbox{d}t$, or equivalently $\displaystyle \frac{\Gamma(x)\Gamma(y)}{\Gamma(x+y)}$ Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) $\displaystyle \frac{\Gamma(x)\Gamma(y)}{\Gamma(x+y)}$. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff_1st(beta)(x,y)` and `diff_2nd(beta)(x,y)` computes the partial diffs of $f$, where $f$ is the function $(x,y) \rightarrow \ \mbox{B}(x,y)$.
