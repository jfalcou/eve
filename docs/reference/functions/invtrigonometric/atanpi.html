**Expressive Vector Engine** `eve::atand`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of atand. Member functions ==================================================================================================== Computes the principal value of the arc cotangent of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cotangent of the input in the range $[-\frac12, \frac12]$. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, $\pm\frac\pi2$ is returned. * If the element is a `Nan`, `NaN` is returned.
