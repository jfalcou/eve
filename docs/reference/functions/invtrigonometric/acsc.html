**Expressive Vector Engine** `eve::acsc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acsc. Member functions ==================================================================================================== Computes the principal value of the arc cosecant of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cosecant of the input in the range $[-\pi/2 , \pi/2]$. In particular: * If the element is $\pm1$, $\pm\frac\pi2$ is returned. * If the element $|x| < 1$, `NaN` is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(acsc)(x)` computes the derivative of the function at `x`.
