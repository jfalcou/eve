**Expressive Vector Engine** `eve::acot`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acot. Member functions ==================================================================================================== Computes the principal value of the arc cotangent of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cotangent of the input in the range $[-\frac\pi2, \frac\pi2]$. In particular: * If the element is $\pm0$, $\pm\frac\pi2$ is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `acot` supports the restricted_, small_, medium_ and big_ decorators.
