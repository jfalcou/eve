**Expressive Vector Engine** `eve::acscpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acscpi. Member functions ==================================================================================================== Computes the principal value of the arc cosecant of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cosecant in degrees of the input in the range $[-\frac12, \frac12]$. In particular: * If the element is $\pm1$, $\pm\frac12$ is returned. * If the element $|x| < 1$, `NaN` is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `acscpi` supports the restricted_, small_, medium_ and big_ decorators.
