**Expressive Vector Engine** `eve::acotd`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acotd. Member functions ==================================================================================================== Computes the principal value of the arc cotangent of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cotangent of the input in the range $]-90, 90]$. In particular: * If the element is $\pm0$, $\pm90$ is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `acotd` supports the restricted, small, medium and big decorators.
