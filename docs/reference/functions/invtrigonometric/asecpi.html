**Expressive Vector Engine** `eve::asecpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of asecpi. Member functions ==================================================================================================== Computes the principal value of the arc secant of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc secant of the input in $\pi$ mutiples in the range $[0, 1]$. In particular: * If the element is $1$, $+0$ is returned. * If the element is $0$, $1$ is returned. * If the element $|x| < 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(asecpi)(x)` computes the derivative of the function at `x`.
