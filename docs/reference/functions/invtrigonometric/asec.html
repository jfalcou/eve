**Expressive Vector Engine** `eve::asec`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of asec. Member functions ==================================================================================================== Computes the principal value of the arc secant of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc secant of the input in the range $[0 , \pi]$. In particular: * If the element is $1$, $+0$ is returned. * If the element is $0$, $\pi$ is returned. * If the element $|x| < 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `cos` supports the restricted, small, medium and big decorators.
