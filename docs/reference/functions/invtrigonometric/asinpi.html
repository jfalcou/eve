**Expressive Vector Engine** `eve::asinpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of asinpi. Member functions ==================================================================================================== Computes the principal value of the arc sine of `x` in degrees. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc sine of the input in the range $[-\frac12, \frac12]$. In particular: * If the element is $1$, $+0$ is returned. * If the element $|x| > 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(asinpi)(x)` computes the derivative of the function at `x`.
