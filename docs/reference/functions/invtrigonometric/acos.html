**Expressive Vector Engine** `eve::acos`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acos. Member functions ==================================================================================================== Computes the principal value of the arc cosine of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cosine of the input in the range $[0 , \pi]$. In particular: * If the element is $1$, $+0$ is returned. * If the element $|x| > 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`raw`](../decorators.html#raw) : The call `raw(rsqrt)(x)`, call a faster implementation which can be slightly less accurate near $\pi$.. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(acos)(x)` computes the derivative of the function at `x`.
