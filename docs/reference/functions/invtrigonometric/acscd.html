**Expressive Vector Engine** `eve::acscd`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acscd. Member functions ==================================================================================================== Computes the principal value of the arc coseccant of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cosecant in degrees of the input in the range $[-90, 90]$. In particular: * If the element is $\pm1$, $\pm90$ is returned. * If the element $|x| < 1$, `NaN` is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `Nan`, `NaN` is returned.
