**Expressive Vector Engine** `eve::asind`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of asind. Member functions ==================================================================================================== Computes the principal value of the arc sine of `x` in degrees. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc sine of the input in the range $[-90, 90]$. In particular: * If the element is $1$, $+0$ is returned. * If the element $|x| > 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `asind` supports the restricted_, small_, medium_ and big_ decorators.
