**Expressive Vector Engine** `eve::acospi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acospi. Member functions ==================================================================================================== Computes the principal value of the arc cosine of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) value in $pi$ multiples of the arc cosine of the input in the range $[0, 1]$. In particular: * If the element is $1$, $+0$ is returned. * If the element $|x| > 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(acospi)(x)` computes the derivative of the function at `x`.
