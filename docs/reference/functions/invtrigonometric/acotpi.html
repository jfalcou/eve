**Expressive Vector Engine** `eve::acotpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acotpi. Member functions ==================================================================================================== Computes the principal value of the arc cotangent of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc cotangent of the input in the range $]-\frac12, \frac12]$. In particular: * If the element is $\pm0$, $\pm\frac12$ is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(acotpi)(x)` computes the derivative of the function at `x`.
