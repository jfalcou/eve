**Expressive Vector Engine** `eve::asin`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of asin. Member functions ==================================================================================================== Computes the principal value of the arc sine of `x`. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) arc sine of the input in the range $[-\frac\pi2 , \frac\pi2]$. In particular: * If the element is $1$, $+0$ is returned. * If the element $|x| > 1$, `NaN` is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(asin)(x)` computes the derivative of the function at `x`.
