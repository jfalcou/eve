**Expressive Vector Engine** `eve::nenr`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the nenr operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) nearest neigbour interpolation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `t` : [loating_values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- The call `nenr(x, y, t)` is similar to `(t<0.5) ? x : y`. The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. (insert ../../src/nenr.src.html here) (insert ../../out/nenr.out.html here) Supported Decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : The call `pedantic(nenr)(x,y,z)` return nan for `t` outside of $[0,1]$ (i.e. no extrapolation). [`numeric`](../decorators.html#numeric) : The call `numeric(nenr)(x,y,z)` ensures that if one of `x`, `y` is not nan it will be returned anyway.
