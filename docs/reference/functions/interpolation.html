**Expressive Vector Engine** (insert ../../crumbs.html here) # Arithmetic Functions These functions provide scalar and SIMD version of basic interpolation functions. To keep the results homogeneous and coherent between SIMD and scalar mode, the types usable in those functions are required to satisfy [`eve::compatible_values`](../concepts/compatible.html). Ranges and precision issues are documented in a per function basis. ## Functions **Convenience header:** [`eve::cbrp`](interpolation/cbrp.html) : computes the cubic Hermite interpolant [`eve::lerp`](interpolation/lerp.html) : computes the linear interpolant [`eve::nenr`](interpolation/nenr.html) : computes the nearest neighbour interpolant ----------------------------------------------------------------------------------------------------
