**Expressive Vector Engine** (insert ../../crumbs.html here) # Arithmetic Functions These functions provide scalar and SIMD version of basic arithmetic functions. To keep the results homogeneous and coherent between SIMD and scalar mode, the types usable in those functions are required to satisfy [`eve::compatible_values`](../concepts/compatible.html). Ranges and precision issues are documented in a per function basis. ## Functions **Convenience header:** [`eve::abs`](arithmetic/abs.html) : computes the absolute value of its parameter ($|x|$) [`eve::arg`](arithmetic/arg.html) : computes the argument of its parameter [`eve::average`](arithmetic/average.html) : computes the average of its parameters [`eve::cbrt`](arithmetic/cbrt.html) : computes the cubic root of its parameter ($\sqrt[3]{x}$) [`eve::clamp`](arithmetic/clamp.html) : clamps a value between a pair of boundary values [`eve::conj`](arithmetic/conj.html) : computes the complex conjugate of its parameter ($\overline{z}$) [`eve::copysign`](arithmetic/copysign.html) : copies the sign of one of its parameter onto the other [`eve::dist`](arithmetic/dist.html) : computes the distance between its parameters [`eve::fdim`](arithmetic/fdim.html) : computes the positive difference of two floating point values ($max(0,x-y)$) [`eve::fma`](arithmetic/fma.html) : computes the multiplication-addition of its parameters without intermediate roundings if possible ($a \times b+c$) [`eve::fms`](arithmetic/fms.html) : computes the multiplication-substraction of its parameters without intermediate roundings if possible ($a \times b+c$) [`eve::fnma`](arithmetic/fnma.html) : computes the negated multiplication-addition of its parameters without intermediate roundings if possible ($c-a \times b$) [`eve::fnms`](arithmetic/fnms.html) : computes the negated multiplication-substraction of its parameters without intermediate roundings if possible ($-a*b-c$) [`eve::frac`](arithmetic/frac.html) : computes the fractional part of its parameter [`eve::hypot`](arithmetic/hypot.html) : computes the $L_2$ norm of its parameters ($\sqrt{x^2 + y^2}$) [`eve::manhattan`](arithmetic/manhattan.html) : computes the $L_1$ norm of its parameters ($|x| + |y|$) [`eve::max`](arithmetic/max.html) : returns the greater of its parameters [`eve::maxmag`](arithmetic/maxmag.html) : returns the value with the greater absolute value among its parameters [`eve::min`](arithmetic/min.html) : returns the smaller of its parameters [`eve::minmag`](arithmetic/minmag.html) : returns the value with the smaller absolute value among its parameters [`eve::modf`](arithmetic/modf.html) : breaks a number into integer and fractional parts [`eve::negate`](arithmetic/negate.html) : computes the product of its parameter by the sign of the second one [`eve::negatenz`](arithmetic/negatenz.html) : computes the product of its parameter by the sign of the second one while considering the sign of 0 to be one [`eve::oneminus`](arithmetic/oneminus.html) : computes the difference between 1 and its parameter without lose of precision ($1 - x$) [`eve::rec`](arithmetic/rec.html) : computes the inverse of its parameter ($\frac{1}{x}$) [`eve::rsqrt`](arithmetic/rsqrt.html) : computes the inverse square root of its parameter ($\frac{1}{\sqrt{x}}$) [`eve::saturate`](arithmetic/saturate.html) : computes the saturation of its parameter within the domain of a given type [`eve::sign`](arithmetic/sign.html) : computes the sign of its parameter [`eve::signnz`](arithmetic/signnz.html) : computes the sign of its parameter while considering the sign of 0 to be one [`eve::sqr`](arithmetic/sqr.html) : computes the square of its parameter ($x^2$) [`eve::sqr_abs`](arithmetic/sqr_abs.html) : computes the square root of the absolute value of its parameter ($|x|^2$) [`eve::sqrt`](arithmetic/sqrt.html) : computes the square root of its parameter ($\sqrt{x}$) ----------------------------------------------------------------------------------------------------
