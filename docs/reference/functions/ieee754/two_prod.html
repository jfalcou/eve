**Expressive Vector Engine** `eve::two_prod`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the two_prod operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) arithmetic two_prod of its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- computes [element-wise](../../../glossary.html#element-wise_) a pair of values `[a,e]` such that: * `a` is `x*y` * `e` is a value such that `a`$\oplus$`e` is equal to `x`$\otimes$`y` where $\oplus$ (resp. $\otimes$) multiplies (resp. adds ) its two parameters with infinite precision.
