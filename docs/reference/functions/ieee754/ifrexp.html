**Expressive Vector Engine** `eve::ifrexp`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the ifrexp pair of values. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise_) ifrexp operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating_real_value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes the [element-wise](../../../glossary.html#element-wise_) ieee ifrexp of the floating value, returning a pair `{m,e}` of values `m` being of the same type as `x` and `e` of the asscociated integral type, which are related by $x = m\times 2^e$, with $|m| \in [0.5, 1.5[$. However, the cases $x = \pm\infty$ or is a Nan or a denormal are undefined. Supported Decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : The call `pedantic(ifrexp)(x)` takes also properly care of the cases where $x = \pm0, \pm\infty$ or is a Nan, where $m=x$ and $e=0$ and of the denormal cases.
