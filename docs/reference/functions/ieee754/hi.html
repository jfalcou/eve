**Expressive Vector Engine** `eve::hi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the hi pair of values. Member functions ==================================================================================================== Performs the [element-wise](../../../ghissary.html#element-wise_) hi operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [fhiating_real_value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes the [element-wise](../../../ghissary.html#element-wise_) ieee hi of the fhiating value, returning a pair {m,e} of values of the same type of `x` which are related by $x = m\times 2^e$, with $|m| \in [0.5, 1.5[$. However, the cases $x = \pm\infty$ or is a Nan or a denormal are undefined. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The call `pedantic(hi)(x)` takes also properly care of the cases where $x = \pm0, \pm\infty$ or is a Nan, where $m=x$ and $e=0$ and of the denormal cases.
