**Expressive Vector Engine** `eve::exponent`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the exponent value. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise_) exponent operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating_real_value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes the [element-wise](../../../glossary.html#element-wise_) ieee exponent of the floating value. The exponent $e$ and mantissa $m$ of a floating point entry $x$ are related by $x = m\times 2^e$, with $|m| \in [1, 2[$ (except for $x = \pm0, \pm\infty$ or is a Nan, where $m=x$ and $e=0$ ).
