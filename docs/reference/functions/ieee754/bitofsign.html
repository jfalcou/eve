**Expressive Vector Engine** `eve::bitofsign`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the bitofsign value. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise_) bitofsign operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- The call `bitofsign(x)` is semantically equivalent to `x && signmask(as(x))`. !!! WARNING Take care that for [floating real values](../../concepts.html#value) bitofsign does NOT return a [logical value](../../concepts.html#value) that can be tested, but `mzero(as(x))` if `x` is negative and `zero(as(x))` if `x` is positive, which both satisfy the `is_eqz` predicate.
