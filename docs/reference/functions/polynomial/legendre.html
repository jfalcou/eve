**Expressive Vector Engine** `eve::legendre`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the legendre operation. Member functions ==================================================================================================== Performs the computation of $\displaystyle P_n(x) = \frac1{2^n n!}\frac{d^n}{dx^n}((x^2-1)^n)$ Parameters ---------------------------------------------------------------------------------------------------- `n` : [integral valuex](../../concepts.html#value). `x` : [floating valuex](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the value of the first kind Legendre polynomial of order `n` at `x`. By choice returns a nan if x is outside of $[-1,1]$. The result type is of the compatibility type of the parameters. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(legendre)(n,x)` computes the derivative of the function at `x`.
