**Expressive Vector Engine** `eve::gegenbauer`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the gegenbauer operation. Member functions ==================================================================================================== Performs the computation of ${\displaystyle C_{n}^{(\lambda )}(z)={\frac {(2\lambda )^{\underline {n}}}{n!}}\,_{2}F_{1}\left(-n,2\lambda +n;\lambda +{\frac {1}{2}};{\frac {1-z}{2}}\right)}$ where $\underline {n}$ is the decreasing factorial $\left(\displaystyle x^{\underline {n}}={\frac {x!}{(x-n)!}}\right)$. Parameters ---------------------------------------------------------------------------------------------------- `n` : [integral values](../../concepts.html#value). `l` : [floating values](../../concepts.html#value), elements must be greater than $-\frac12$. `x` : [floating values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the value of the gegenbauer polynomial of order `n` at `l`, `x`: The result type is of the compatibility type of the parameters. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(gegenbauer)(n,l,x)` computes the derivative of the function at `x`.
