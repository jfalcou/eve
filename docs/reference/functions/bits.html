**Expressive Vector Engine** These functions provide scalar and SIMD version of bit operations. Bitwise operations are extended to work on any :ref:`concept-value` (including :ref:`concept-ieeevalue`) types by considering the masking operations to apply on the value bit patterns while retaining its original types. There is no type nor cardinal restrictions between operands of bitwise function as long as their size in bytes are equal. This implies calls of such function on :ref:`Ieee Values ` are possible as long as they are performed with a second parameters of proper size. The header `eve/function/bit.hpp` allows the inclusion of all specific bit functions.
