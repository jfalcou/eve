**Expressive Vector Engine** `eve::fls`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the fls operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) fls operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [unsigned value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#element-wise_) the index or position of the last significant bit set to one in the word counting from the least significant bit position. The result is undefined for a zero input if the pedantic decorator is not used. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The expression `pedantic(fls)(x)` returns the size of the element type (an out of range index) if the input is zero.
