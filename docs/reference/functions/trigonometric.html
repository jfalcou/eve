**Expressive Vector Engine** Trigonometric Functions
These functions provide scalar and SIMD version of the direct trigonometric operations in radians, degrees and $\pi$ multiples. Each one is supported with four decorators aside the regular call, which all provides the same accuracy, but in different ranges and this is peculiarly important for radian and degree inputs: * Radians The possible decorators are: - restricted is the fastest but gives the correct result in $[-\pi/4, +\pi/4]$ only and Nan outside. - small gives the correct result for $[-\pi/2, +\pi/2]$ only and Nan outside. - medium gives the correct result for $|x| < 536870912.0f$ (float) or $ |x| < 2.0e14 $ (double) - big gives the correct result in the whole range. Without any decorator the call tests the input value to choose among the decorated ones the best fit. Of course, with SIMD parameter the fit is the interval containing all the vector elements. The rationale to provide these flavours is that the more costly part of the computation of a trigonometric function from the radian angle is the reduction of the argument modulo $\pi/4$ that, to be correct in the full range must use a few hundreds of $\pi$ decimals: - if $x \in [-\pi/4, +\pi/4]$ there is no reduction to perform. - if $x \in [-\pi, +\pi]$ the work is not a lot heavier. - if $x \in $ a simplified scheme can be used. - for the whole range a quite heavy algorithm is to be used. The time ratio between the fastest and the slowest can be of 20. If you are not sure of your interval the regular call works and the penalty is the range test that is not much in the medium and big cases, but not neglectable in the restricted and small ones. * Degrees For degree the functions are computed by multiplying the entry by $\pi/180$ and so the same comments are valid on the same intervals expressed in degrees. Also the multiplication causes a little loss in accuracy as $\pi$ is not represented exactly. * $\pi$ multiples In this case the reduction is a lot less expansive, because it's quite just a truncation to perform, and the the various flavour (except restricted) are just provided for orthogonality.
