Member functions ==================================================================================================== Performs the computation of the cauchy distribution attributes Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). `m` : [floating real value](../../concepts.html#value). median of the distribution (default to 0) `s` : [floating real value](../../concepts.html#value). scale (>0) of the distribution (default to 1) Return value ---------------------------------------------------------------------------------------------------- The regular call computes the cdf of the cauchy distribution of median `m` and scale `s` at `x`. In partucular: Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) The expression `diff_nth< N >(cauchy)(x,m,s)` computes the partial derivatives of the cdf function relative to the Nth parameter.
