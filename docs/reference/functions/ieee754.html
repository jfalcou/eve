**Expressive Vector Engine** (insert ../../crumbs.html here) # IEE754 Functions These functions provide scalar and SIMD version of functions related to the manipulation of IEEE754 representation of floating numbers. ## Functions **Convenience header:** [`eve::bitofsign`](ieee754/bitofsign.html) : extracts the bit of sign of its parameter [`eve::exponent`](ieee754/exponent.html) : extracts the exponent of its parameter [`eve::frexp`](ieee754/frexp.html) : breaks a number into significand and a power of 2 [`eve::ifrexp`](ieee754/ifrexp.html) : breaks a number into significand and an integral power of 2 [`eve::ldexp`](ieee754/ldexp.html) : multiplies a number by 2 raised to a power [`eve::lo`](ieee754/lo.html) : return the lower half of the input [`eve::lohi`](ieee754/lohi.html) : return the lower half and higher half of the input [`eve::hi`](ieee754/hi.html) : return the higher half of the input [`eve::mantissa`](ieee754/mantissa.html) : extracts the mantissa of its parameter [`eve::nb_values`](ieee754/nb_values.html) : computes the number of IEEE754 values between its parameters [`eve::nextafter`](ieee754/nextafter.html) : determines next representable floating-point value towards the given value [`eve::ulpdist`](ieee754/ulpdist.html) : computes the distance in ULP between its parameters ----------------------------------------------------------------------------------------------------
