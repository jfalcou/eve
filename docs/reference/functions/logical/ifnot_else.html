(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the ifnot_else operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) ifnot_else operation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `z` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- The call `ifnot_else(x, y, z)` is performs a choice between the elements of `y` and `z` according to the truth value of the elements of `x`.
