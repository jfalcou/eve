**Expressive Vector Engine** `eve::logical_andnot`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the logical ANDNOT operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular_) logical ANDNOT of the two [values](../../concepts.html#value). Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes logical ANDNOT of the two parameters following the [logical operations semantic](../../../glossary.html#Logical). the call `logical_andnot(x, y)` is semantically equivalent to `x && !y` if `x` or `y` is an [simd value](../../concepts.html#value). !!! WARNING Although the infix notation with `&&` and `!` is supported for all [simd values](../../concepts.html#value), the `&&` operator on standard scalar type is the original one and so does not allow floating parameters. Moreover if a proper intrinsic exists for ANDNOT the infix notation will not be able use it.
