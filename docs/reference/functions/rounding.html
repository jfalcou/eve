**Expressive Vector Engine** Rounding Functions
The following functions provide scalar and SIMD version of rounding to integer functions. For integral inputs they all act as identity, otherwise provide access to the common rounding rules. Namely: rounding to nearest, toward plus or minus infinity or truncation towards zero.
