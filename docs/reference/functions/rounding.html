**Expressive Vector Engine** (insert ../../crumbs.html here) # Rounding Functions The following functions provide scalar and SIMD version of rounding to integer functions. For integral inputs they all act as identity, otherwise provide access to the common rounding rules. ## Functions **Convenience header:** [`eve::ceil`](rounding/ceil.html) : computes the smallest integer not less than its parameter [`eve::floor`](rounding/floor.html) : computes the largest integer not greater than its parameter [`eve::nearest`](rounding/nearest.html) : computes the closest integer to its parameter, rounding downward [`eve::round`](rounding/round.html) : rounds to an integer using a given rounding mode [`eve::trunc`](rounding/trunc.html) : rounds to nearest integer not greater in magnitude than its parameter ----------------------------------------------------------------------------------------------------
