**Expressive Vector Engine** `eve::cbrp`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the cbrp operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) cubic interpolation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `xp`, `yp`, `t` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- The call `cbrp(x, xp, y, yp,t)` computes elementwise at `t` the value of the cubic polynomial $p$ such that $p(0)=x$, $p(1)=y$, $p^\prime(0)=xp$ and $p^\prime(1)=yp$. The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. When `x` and `y` are finite: - If `t == 0`, the result is equal to a. - If `t == 1`, the result is equal to b. - If `t >= 0` and `t <= 1`, the result is finite but generally of no use. Supported Decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : The call `pedantic(cbrp)(x,y,z)` return nan for `t` outside of $[0,1]$ (i.e. no extrapolation).
