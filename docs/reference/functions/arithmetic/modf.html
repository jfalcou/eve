**Expressive Vector Engine** `eve::modf`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the modf operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) modf operation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- A pair of values with the same type as `x` containing respectively the [element-wise](../../../glossary.html#value) fractional and integral parts of `x`, each having the type and sign of `x`. In particular: * If `x` is infinite `{Nan, x}` is returned. * If `x` is a `Nan` `{Nan, Nan}` is returned. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The call `pedantic(modf)(x)` ensures standard conformity : if `x` is infinite, `{0, x}` is returned.
