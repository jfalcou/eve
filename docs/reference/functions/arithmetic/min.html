**Expressive Vector Engine** `eve::min`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the min operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) min operation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `args`, ... : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- For each tuple of corresponding [`elements`](../../../glossary.html#element-wise), the smaller is returned. The result type is the [compatibility result](../../concept.html#compatibility) of the parameters. !!! WARNING If any element of the inputs is a `Nan`, the corresponding output element is system-dependent. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The call `pedantic(min)(x, y)` ensures the conformity to the standard behaviour, that is (on an [element-wise](../../../glossary.html#value) basis) semanticaly equivalent to: `(x < y) ? x : y` and this behaviour is also ensured on n parameters calls as if this scheme was recursively used. [`numeric`](../decorators.html#numeric) : **Required header: ** : The call `numeric(min)(x,y)` ensures that if any element of the inputs is a `Nan`, the corresponding output element is the corresponding element of the other parameter (i.e. the not invalid parameter wins) and this behaviour is also ensured on n parameters calls as if this scheme was recursively used. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative< N >(min)(x,args,...)` computes the partial derivative relative to the Nth parameter. If the parameters are $x_1, ..., x_n$ and their minimum is $m$, the value returned is elementwise 1 if $m$ is equal to $x_N$ else 0.
