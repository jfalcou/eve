**Expressive Vector Engine** `eve::min`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the min operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) min operationr on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- For each pair of corresponding [`elements`](../../../glossary.html#element-wise), the smaller is returned. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. !!! WARNING If any element of the inputs is a `Nan`, the corresponding output element is system-dependent. supported Decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : The call `pedantic(min)(x,y)` ensures the conformity to the standard behaviour, that is (on an [element-wise](../../../glossary.html#value) basis) semanticaly equivalent to: `(x < y) ? x : y`. [`numeric`](../decorators.html#numeric) : The call `numeric(min)(x,y)` ensures that if any element of the inputs is a `Nan`, the corresponding output element is the corresponding element of the other parameter (i.e. the not invalid parameter wins).
