**Expressive Vector Engine** `eve::lerp`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the lerp operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) linear interpolation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `t` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- The call `lerp(x, y, t)` is similar to `(1-t)*x+t*y`. The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. When `x` and `y` are finite: - If `t == 0`, the result is equal to a. - If `t == 1`, the result is equal to b. - If `t >= 0` and `t <= 1`, the result is finite. Supported Decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : The call `pedantic(lerp)(x,y,z)` return nan for `t` outside of $[0,1]$ (i.e. no extrapolation). [`numeric`](../decorators.html#numeric) : The call `numeric(lerp)(x,y,z)` ensures that: - If `t` is finite and `x == y`, the result is equal to `x`. - If `t` is finite or `(!isnan(t) && x-y != 0)`, the result is not NaN.
