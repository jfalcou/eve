Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#value) positive difference between `x` and `y`: * if `x>y`, x-y is returned * if `x<=y`, +0 is returned * otherwise a `Nan` is returned Supported decorators ==================================================================================================== [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative_1st(fim)(x,y)` and `derivative_2nd(fim)(x,y)` computes the partial derivatives of $f$, where $f$ is the function $(x,y) \rightarrow \ \max(0,x-y)$.
