**Expressive Vector Engine** `eve::oneminus`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the oneminus operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) oneminus operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) `1-x` with the type of `x`. !!! WARNING If an [element](../../../glossary.html#element_) of the expected result is not representable in the result type, the corresponding result [element](../../../glossary.html#element_) is undefined. Supported decorators ==================================================================================================== [`saturated`](../decorators.html#saturated) : **Required header: ** : The call `saturated(oneminus)(x)` is semantically equivalent to `saturated(sub)(One(as(x)), x)` and is never undefined. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(oneminus)(x)` computes the derivative of the function at `x`.
