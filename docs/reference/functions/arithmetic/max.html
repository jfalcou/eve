**Expressive Vector Engine** `eve::max`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the max operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) max operation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `args` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- For each tuple of corresponding [`elements`](../../../glossary.html#element-wise), the greater is returned. The result type is the [compatibility result](../../concept.html#compatibility) of the parameters. !!! WARNING If any element of the inputs is a `Nan`, the corresponding output element is system-dependent. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The call `pedantic(max)(x,args,...)` ensures the conformity to the standard behaviour, that is for two parameters (on an [element-wise](../../../glossary.html#value) basis) to be semanticaly equivalent to: `(x < y) ? y : x` and this behaviour is also ensured on n parameters calls as if this scheme was recursively used. [`numeric`](../decorators.html#numeric) : **Required header: ** : The call `numeric(max)(x,args,...)` ensures that if any element of the inputs is not a `Nan`, the corresponding output element will not be a `Nan`. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative< N >(max)(x,args,...)` computes the partial derivative relative to the Nth parameter. If the parameters are $x_1, ..., x_n$ and their maximum is $m$, the value returned is 1 if $m$ is equal to $x_N$ and otherwise 0.
