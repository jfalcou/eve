**Expressive Vector Engine** `eve::max`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the max operation. Member functions ==================================================================================================== Performs the max operator on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- On an [element-wise](../../../glossary.html#elment-wise) basis, the greater of `x` and `y`. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. !!! WARNING If any element of the inputs is a `Nan`, the corresponding output element is system-dependent. supported Decorators ==================================================================================================== [`pedantic_`](../decorators.html#pedantic_) : The call `pedantic_(max)(x,y)` ensures the conformity to the standard behaviour, that is (on an [element-wise](../../../glossary.html#value) basis) semanticaly equivalent to: `(x < y) ? y : x`. [`numeric_`](../decorators.html#numeric_) : The call `numeric_(max)(x,y)` ensures that if any element of the inputs is a `Nan`, the corresponding output element is the corresponding element of the other parameter (i.e. the not invalid parameter wins).
