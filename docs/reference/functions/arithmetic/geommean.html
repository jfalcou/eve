**Expressive Vector Engine** `eve::geommean`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the mid-point operation. Member functions ==================================================================================================== Computes the [element-wise](../../../glossary.html#regular) geometric mean of the [values](../../concepts.html#value). Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` or `x`, `args`, ... : [floating values](../../concepts.html#value). Return value ------------------- --------------------------------------------------------------------------------- For two parameters $\sqrt{xy}$. No overflow occurs. For more than two parameters $\sqrt[n]{x_1x_2...x_n}$ The result type is the [compatibility result](../../concept.html#compatibility) of the parameters. !!! WARNING if the number of parameters is even and the product of the parameters is srictly negative the result is a Nan. if the number of parameters is odd the result has the sign of the product of the parameters Supported decorators ==================================================================================================== [`diff_nth`](../decorators.html#diff) : **Required header: ** : The expression `diff_nth< N >(geommean)(x,args...)` computes the partial diff of the function relative to its Nth parameter. The returned value is 0 if N is greater that the actual number of parameters, otherwise it is similar to `geommean(x1,x2,...,xn)/(n*xN)`.
