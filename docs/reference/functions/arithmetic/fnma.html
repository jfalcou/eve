**Expressive Vector Engine** `eve::fnma`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the fused negate-multiply-add operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) fused negate-multiply-add operation on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `z` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- The call `fnma(x, y, z)` is similar to `-x*y+z` as if calculated to infinite precision and rounded once to fit the result as much as supported by the hardware. The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. !!! WARNING Note This `fnma` implementation provides those properties for all [real integral values](../../concepts.html#integral_value) and when possible for [real floating values](../../concepts.html#value). Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The call `pedantic(fnma)(x,y,z)` ensures the one rounding property. This can be very expensive if the system has no hardware capability. [`numeric`](../decorators.html#numeric) : **Required header: ** : The call `numeric(fnma)(x,y,z)` ensures the full compliance to fnma properties. This can be very expensive if the system has no hardware capability. [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff_1st(fnma)(x,y,z)`, `diff_2nd(fnma)(x,y,z)` and `diff_3rd(fnma)(x,y,z)` computes the partial diffs of $f$, where $f$ is the function $(x,y,z) \rightarrow \ -xy+z$.
