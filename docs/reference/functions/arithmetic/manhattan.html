**Expressive Vector Engine** `eve::manhattan`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the manhattan operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) manhattan operation on its parameters. The value computed is the $l_1$ norm in $\mathbb{R}^2$ or $\mathbb{R}^3$ respectively. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `z` : [value](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- For each [`element`](../../../glossary.html#element-wise) of the input: 1. $|x|+|y|$ is returned. 2. $|x|+|y|+|z|$ is returned. The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** : The call `pedantic(manhattan)(x,y)` or `pedantic(manhattan)(x,y,z)` computes the sum of the absolute values of its parameters returning $\infty$ as soon as one of its parameter is infinite, regardless of possible `Nan` values. [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative_1st(manhattan)(x,y)` and `derivative_2nd(manhattan)(x,y)` computes the partial derivatives of $f$, where $f$ is the function $(x,y) \rightarrow \ |x|+|y|$. Of course 3 analogous calls can be made in the three parameters case.
