**Expressive Vector Engine** `eve::minmag`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the minmag operation. Member functions ==================================================================================================== Performs the minmag [element-wise](../../../glossary.html#regular) operator on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- For each tuple of corresponding [`elements`](../../../glossary.html#element-wise), the element of the least absolute value is returned. For instance for two elemnts: * If `|x| > |y|`, `y` is returned. * If `|x| < |y|`, `x` is returned. * Otherwise `min(x, y)` is returned. The result type is the [compatibility result](../../concept.html#compatibility) of the parameters. !!! WARNING If any element of the inputs is a `Nan`, the corresponding output element is system-dependent. Supported decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : **Required header: ** The behaviour is the same except that if `|x| == |y|`, `pedantic(min)` is used. [`numeric`](../decorators.html#numeric) : **Required header: ** The behaviour is the same except that if `|x| == |y|`, `numeric(min)` is used. [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff< N >(minmag)(x,args,...)` computes the partial diff relative to the Nth parameter. If the parameters are $x_1, ..., x_n$ and their minmag is $m$, the value returned is elementwise 1 if $m$ is equal to $x_N$ else 0.
