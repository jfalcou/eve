abs ==================================================================================================== **Required header:** **Synopsis:** Function object representing the computation of the absolute value **Members function:** Performs the element-wise absolute value on its parameter. **Parameters:** `x` : value of :ref:`value ` types. **Return value:** A value with the same type as `x` containing the absolute value of `x` if its representable in this type. More specifically, for signed integers the absolute value of :ref:`Valmin ` is not representable and the result is undefined. !!! WARNING `abs` is also a standard library function name and there possibly exists a C macro version which may be called instead of the **EVE** version. To avoid this, use the `eve::abs` notation. **Supported Decorators:** [`saturated`](../../decorators.html#saturated) : The expression `saturated_(abs)(x)` computes a saturated absolute value of `x`. This guarantees that the result is equal or greater than 0. More specifically, for any signed integer-like type `iT`, the expression `saturated_(abs)(Valmin(as()))` returns `eve::Valmax( as() )`.
