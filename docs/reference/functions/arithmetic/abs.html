eve::abs ==================================================================================================== **Required header:** **Synopsis:** Function object representing the computation of the absolute value **Member functions:** Performs the element-wise absolute value on its parameter. **Parameters:** `x` : value of :ref:`value ` types. **Return value:** A value with the same type as `x` containing the absolute value of `x` if its representable in this type. More specifically, for signed integers the absolute value of [their lowest finite value](../../constants.html#valmin) is not representable and the result is undefined. !!! WARNING `abs` is also a standard library function name and there possibly exists a C macro version which may be called instead of the **EVE** version. To avoid this, use the `eve::abs` notation. **Supported Decorators:** [`saturated_`](../../decorators.html#saturated_) : The expression `saturated_(abs)(x)` computes a saturated absolute value of `x`. This guarantees that the result is greater or equal than 0. More specifically, for any signed integer value `x`, the expression: `saturated_(abs)(Valmin(as(x)))` evaluates to: `eve::Valmax(as(x))`.
