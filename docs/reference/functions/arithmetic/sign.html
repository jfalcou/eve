**Expressive Vector Engine** `eve::sign`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the sign operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) sign operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) the sign of `x`. For [real value](../../concepts.html#value) `x` is semantically equivalent to: * If x is greater than 0, 1 is returned. * If x is less than 0, -1 is returned. * If x is zero, x is returned.
