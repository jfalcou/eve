**Expressive Vector Engine** `eve::hypot`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the hypot operation. Member functions ==================================================================================================== Performs the hypot operator on its parameters. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value) `y` : [value](../../concepts.html#value) `z` : [value](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- * The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. * On an [element-wise](../../../glossary.html#elment-wise) basis the function computes: - With two parameters: $\sqrt{x^2+y^2}$ - With three parameters: $\sqrt{x^2+y^2+z^2}$ * The [`regular_`](../decorators.html#regular_) call uses the naive formula that may easily overflow. supported Decorators ==================================================================================================== [`pedantic_`](../decorators.html#pedantic_) : The call `pedantic_(hypot)(x,y)` or `pedantic_(hypot)(x,y,z)` computes the square root of the sum of the squares of x and y, (and z) without undue overflow or underflow at intermediate stages of the computation and can be more accurate than the [`regular_`](../decorators.html#regular_) call.
