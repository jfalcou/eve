**Expressive Vector Engine** `eve::hypot`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the hypot operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) hypot operation on its parameters. The value computed is the $l_2$ euclidian norm in $\mathbb{R}^2$ or $\mathbb{R}^3$ respectively. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `z` : [value](../../concepts.html#value) Return value ---------------------------------------------------------------------------------------------------- For each [`element`](../../../glossary.html#element-wise) of the input: 1. $\sqrt{x^2+y^2}$ is returned. 2. $\sqrt{x^2+y^2+z^2}$ is returned. The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. Supported Decorators ==================================================================================================== [`pedantic`](../decorators.html#pedantic) : The call `pedantic(hypot)(x,y)` or `pedantic(hypot)(x,y,z)` computes the square root of the sum of the squares of `x` and `y`, (and `z`) without undue overflow or underflow at intermediate stages of the computation and can be more accurate than the [`regular`](../decorators.html#regular) call. Morever it returns $\infty$ as soon as one of its parameter is infinite, regardless of possible `Nan` values.
