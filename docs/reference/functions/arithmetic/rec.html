**Expressive Vector Engine** `eve::rec`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the rec operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) rec operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) the inverse of `x`. For [real integral value](../../concepts.html#value) `x` is semantically equivalent to: * If x==1 or x==-1, x is returned. * If x==0, [the greatest representable positive value](../../constants.html#valmax) is returned. * Otherwise 0 is returned. Supported Decorators ==================================================================================================== [`raw`](../decorators.html#raw) : The call `raw(rec)(x)`, call a proper system intrinsic if one exists, but with possibly very poor accuracy in return.
