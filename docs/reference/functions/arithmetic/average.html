**Expressive Vector Engine** `eve::average`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the mid-point operation. Member functions ==================================================================================================== Perform the arithmetic average of the two [values](../../concepts.html#value). Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value) of type T `y` : [value](../../concepts.html#value) of type U Return value ---------------------------------------------------------------------------------------------------- * The result type is the [compatibility result](../../concept.html#compatibility) of the three parameters. * Half the sum of `x` and `y`. No overflow occurs. * If `x` and `y` are [integral values](../../concepts.html#value) type and the sum is odd, the result is a rounded value at a distance guaranteed to be less than or equal to 0.5 of the average floating value, but may differ by unity from the truncation given by `(x+y)/2`. Moreover, as some architectures provide simd intrinsics to perform the operation, it is possible that the scalar results may differ by one unit from simd ones..
