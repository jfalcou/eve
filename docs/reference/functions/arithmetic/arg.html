(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the phase angle (in radians). Member functions ==================================================================================================== The phase angle of `x` in radians. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- * The phase angle with the same type as the parameter. For real input values the result elements are always $0$ or $\pi$ (complex values are not supported yet). * The [`regular_`](../../decorators.html#regular_) call returns the value zero if the bit of sign of the entry element is not set and the value $\pi$ if it is set. Supported Decorators ---------------------------------------------------------------------------------------------------- [`pedantic_`](../../decorators.html#pedantic_) : With this decorator the function returns a Nan element for each corresponding Nan entry element.
