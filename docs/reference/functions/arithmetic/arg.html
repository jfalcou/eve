**Expressive Vector Engine** `eve::arg`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the phase angle (in radians). Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise_) phase angle operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- For each [`element`](../../../glossary.html#element-wise) of the input the phase angle is returned: * if the bit of sign of the entry [element](../../../glossary.html#elment-wise) is not set, $0$ is returned. * if the bit of sign of the entry [element](../../../glossary.html#elment-wise) is set, $\pi$ is returned. Supported decorators ---------------------------------------------------------------------------------------------------- [`pedantic`](../../decorators.html#pedantic) : With this decorator the function returns a Nan element for each corresponding Nan entry element. [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(arg)(x)` computes the derivative of the function at `x`.
