**Expressive Vector Engine** `eve::acsch`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acsch. Member functions ==================================================================================================== Performs $\log(1/x+\sqrt{1/x^2+1})$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) inverse hyperbolic cosine of the input. The inverse hyperbolic sine is semantically equivalent to $\log(1/x+\sqrt{1/x^2+1})$. In particular: * If the element is $\pm\infty$, $\pm0$ returned. * If the element is $\pm1$, $\pm\infty$ is returned. * If the element does not belong $]0,1[$, `NaN` is returned. Supported decorators ==================================================================================================== [`derivative`](../decorators.html#derivative) : **Required header: ** : The expression `derivative(acsch)(x)` computes the derivative of the function at `x`.
