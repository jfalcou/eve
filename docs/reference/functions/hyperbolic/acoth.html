**Expressive Vector Engine** `eve::acoth`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of acoth. Member functions ==================================================================================================== Performs $\frac{1}{2}\log((x+1)/(x-1))$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) inverse hyperbolic cotangent of the input. The inverse hyperbolic sine is semantically equivalent to $\frac{1}{2}\log((x+1)/(x-1))$. In particular: * If the element is $\pm1$, $\pm0$ is returned. * If the element is $\pm\infty$, $\pm1$ is returned. * If the element is less than one or a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(acoth)(x)` computes the derivative of the function at `x`.
