**Expressive Vector Engine** `eve::csch`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of csch. Member functions ==================================================================================================== Performs $\frac2{e^x-e^{-x}}$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) hyperbolic secant of the input. In particular: * If the element is $\pm0$, $\pm\infty$ is returned. * If the element is $\pm\infty$, $\pm0$ is returned. * If the element is a `NaN`, `NaN` is returned.
