**Expressive Vector Engine** `eve::tanh`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of tanh. Member functions ==================================================================================================== Performs $\frac{e^x-e^{-x}}{e^x+e^{-x}}$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) hyperbolic tangent of the input. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, $\pm1$ returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(tanh)(x)` computes the derivative of the function at `x`.
