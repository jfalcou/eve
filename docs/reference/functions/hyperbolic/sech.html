**Expressive Vector Engine** `eve::sech`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sech. Member functions ==================================================================================================== Performs $\frac2{e^x+e^{-x}}$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) hyperbolic secant of the input. In particular: * If the element is $\pm0$, 1 is returned. * If the element is $\pm\infty$, $+0$ is returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(sech)(x)` computes the derivative of the function at `x`.
