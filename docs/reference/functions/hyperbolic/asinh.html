**Expressive Vector Engine** `eve::asinh`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of asinh. Member functions ==================================================================================================== Performs $\log(x+\sqrt{x^2+1})$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) inverse hyperbolic cosine of the input. The inverse hyperbolic sine is semantically equivalent to $\log(x+\sqrt{x^2+1})$. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, $\pm\infty$ returned. * If the element is a `NaN`, `NaN` is returned.
