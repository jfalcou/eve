**Expressive Vector Engine** `eve::sinh`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sinh. Member functions ==================================================================================================== Performs $\frac{e^x-e^{-x}}2$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) hyperbolic sine of the input. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, $\pm\infty$ returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(sinh)(x)` computes the derivative of the function at `x`.
