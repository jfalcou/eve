**Expressive Vector Engine** `eve::cosh`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cosh. Member functions ==================================================================================================== Performs $\frac{e^x+e^{-x}}2$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) hyperbolic cosine of the input. In particular: * If the element is $\pm0$, 1 is returned. * If the element is $\pm\infty$, $+\infty$ is returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(cosh)(x)` computes the derivative of the function at `x`.
