**Expressive Vector Engine** `eve::ellint_rc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the ellint_rc operation. Member functions ==================================================================================================== Performs the computation of the `ellint_rc` function. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [floating values](../../concepts.html#value). This computes the degenerate Carlson's elliptic integral $$\displaystyle R_C(x, y) = \frac12 \int_{0}^{\infty} (t+x)^{-1/2}(t+y)^{-1} \mbox{d}t$$ as described in Carlson, Numerische Mathematik, vol 33, 1 (1979) Parameters `x` and `y` must be strictly positive and `y` non zero. In any other case the result is nan. Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the Carlson's integral. The result type is of the compatibility type of the three parameters. (insert ../../src/ellint_rc.src.html here) (insert ../../out/ellint_rc.out.html here)
