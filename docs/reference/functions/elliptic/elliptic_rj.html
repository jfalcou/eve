**Expressive Vector Engine** `eve::ellint_rc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the ellint_rj operation. Member functions ==================================================================================================== Performs the computation of the `ellint_rj` function. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `z`, `p` : [floating values](../../concepts.html#value). This computes the Carlson's elliptic integral $$\displaystyle R_J(x, y, z, p) = \frac32 \int_{0}^{\infty} (t+p)^{-1}[(t+x)(t+y)(t+z)]^{-1/2}\mbox{d}t$$ as described in Carlson, Numerische Mathematik, vol 33, 1 (1979) Parameters `x`, `y` and `z` must be non negative at most one of them 0, and `p` non zero. In any other case the result is nan. Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the Carlson's integral. The result type is of the compatibility type of the three parameters. (insert ../../src/ellint_rj.src.html here) (insert ../../out/ellint_rj.out.html here)
