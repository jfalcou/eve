**Expressive Vector Engine** `eve::ellint_rf`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the ellint_rf operation. Member functions ==================================================================================================== Performs the computation of the `ellint_rf` function. Parameters ---------------------------------------------------------------------------------------------------- `x`, `y`, `z` : [floating values](../../concepts.html#value). This computes the Carlson's elliptic integral of the first kind $\displaystyle R_F(x, y, z) = \frac12 \int_{0}^{\infty} [(t+x)(t+y)(t+z)]^{-1/2} \mbox{d}t$ as described in Carlson, Numerische Mathematik, vol 33, 1 (1979) parameters must be postive and at most one zero else the result is nan. Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#value) Jacobi amplitude. The result type is of the compatibility type of the three parameters. (insert ../../src/ellint_rf.src.html here) (insert ../../out/ellint_rf.out.html here)
