**Expressive Vector Engine** `eve::am`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the am operation. Member functions ==================================================================================================== Performs the computation of the `am` function. Let $F(\phi,k) = F(\phi \backslash \alpha) = F(\phi | m)$, be Legendre's elliptic function of the first kind with modulus $k$, modular angle $\alpha$ where $k = \sin\alpha$ or parameter $m$ where $m = k^2$, i.e. $\displaystyle F(\phi,k) = \int_0^\phi \frac{d\theta}{\sqrt{1 - k^2 \sin^2\theta}}$ $\displaystyle F(\phi \backslash \alpha) = \int_0^\phi \frac{d\theta}{\sqrt{1 - \sin^2\alpha \sin^2\theta}}$ $\displaystyle F(\phi | m) = \int_0^\phi \frac{d\theta}{\sqrt{1 - m \sin^2\theta}}$ This Jacobi elliptic amplitude function, am, is defined as `am(u,k) = ell_angle(am)(u, alpha) = ell_modulus(am)(u, m)` = $\phi$ where $u = F(\phi,k) = F(\phi \backslash \alpha) = F(\phi | m)$. Parameters ---------------------------------------------------------------------------------------------------- `x`, `u` : [floating values](../../concepts.html#value). `tol` : [scalar floating values](../../concepts.html#value). `u` The first argument of am(u,x) corresponding to the value of the elliptic integral of the first kind $u = F(am(u,x),x)$. `x` The second argument of the amplitude function am(u,x) corresponding to the second argument of the elliptic integral of the first kind $F(\phi,x)$. x may be the parameter (or modulus or modular angle with appropriate decoratoration) In the regular cas (no decorator) the parameter must be in $[-1, 1]$ `tol` is the tolerance on the result accuracy which by defaut is set to the `eps` value of the compatibility type element, Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#value) Jacobi amplitude. The result type is of the compatibility type of the two first parameters. Supported decorators ==================================================================================================== [`ell_modulus`](../decorators.html#ell_modulus) : - the parameter `x` is the 'modulus' and must be in $[0, 1]$. - `ell_modulus(am)(u,x)` is equivalent to `am(u,sqrt(x))`. [`ell_angle`](../decorators.html#ell_modulus) : - the parameter `x` is the 'modular angle' in radian. - `ell_angle(am)(u,x)` is equivalent to `am(u,sin(x))`.
