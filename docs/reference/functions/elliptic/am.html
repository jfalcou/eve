**Expressive Vector Engine** `eve::am`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the am operation. Member functions ==================================================================================================== Performs the computation of the `am` function. Let $F(\phi,k) = F(\phi \backslash alpha) = F(phi | m), be Legendre's elliptic function of the first kind with modulus $k$, modular angle $\alpha$ where $k = \sin\alpha$ or parameter $m$ where $m = k^2$, i.e. $\displaystyle F(\phi,k) = \int_0^\phi \frac{d\theta}{\sqrt{1 - k^2 \sin^2\theta}}$ $\displaystyle F(\phi \backslash \alpha) = \int_0^\phi \frac{d\theta}{\sqrt{1 - \sin^2\alpha \sin^2\theta}}$ $\displaystyle F(\phi | m) = \int_0^\phi \frac{d\theta}{\sqrt{1 - m \sin^2\theta}}$ This Jacobi elliptic amplitude function, am, is defined as $am(u,k) = am(u \backslash alpha) = am(u | m) = phi where $u = F(phi,k) = F(phi \backslash alpha) = F(phi | m)$. The second argument of the amplitude function $am(u,x)$ corresponding to the second argument of the elliptic integral of the first kind $F(phi,x)$. $x$ may be the modulus, modular angle, or parameter depending on the value of a decorator choice. If choice is `modulus`, then x must be between 0 and 1 inclusively, If choice is `k` ` then x must be between -1 and 1 inclusively. Parameters ---------------------------------------------------------------------------------------------------- `x`, u : [floating values](../../concepts.html#value). `tol` : [scalar floating values](../../concepts.html#value). a0 The first argument of am(u,x) corresponding to the value of the elliptic integral of the first kind u = F(am(u,x),x). a1 The second argument of the amplitude function am(u,x) corresponding to the second argument of the elliptic integral of the first kind F(phi,x). x may be the modulus, modular angle, or parameter depending on the value of a2. If a2 = 'm', then x must be between 0 and 1 inclusively and if a2 = 'k', then x must be between -1 and 1 inclusively. a3 tol the tolerance on the result accuracy by defaut Eps(), where T is the type of u a2 arg The mode of the second argument of am(): If a2 = 'k', then x = k, the modulus of F(phi,k). If A2 = 'a', then x = alpha, the modular angle of F(phi \ alpha), alpha in radians. If arg = 'm', then x = m, the parameter of F(phi | m). The value of arg defaults to 'k'. Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) The result type is of the compatibility type of the two first parameters.
