**Expressive Vector Engine** `eve::ellint_2`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the ellint_2 operation. Member functions ==================================================================================================== Performs the computation of the `ellint_2` function. Parameters ---------------------------------------------------------------------------------------------------- `phi, `k` : [floating values](../../concepts.html#value). computes the complete and incomplete elliptic integrals of the second kind : $$\displaystyle F(\phi,k) = \int_0^{\phi} \sqrt{1-k^2\sin^2 t\mbox{d}t}$$ The complete case corresponds to $\phi = \pi/2$. - `k` must verify $k^2\sin^2\phi \le 1$ or the result is nan. - In the complete case this means $|k| \le 1$. In any other case the result is nan. Return value ---------------------------------------------------------------------------------------------------- Returns [element-wise](../../../glossary.html#value) the elliptic integral of the first kind. The result type is of the compatibility type of the three parameters. (insert ../../src/ellint_2.src.html here) (insert ../../out/ellint_2.out.html here)
