**Expressive Vector Engine** (insert ../../crumbs.html here) # combinatorial functions These functions provide scalar and SIMD version of some combinatorial functions in namespace `eve`. Although, these functions are primarily devoted to integral typed entries, they work fine on floating ones provided they are flint (floating point representation of integers) and they can admit decorators that will enforce rounding of the parameters. ## Functions **Convenience header:** [`eve::double_factorial`](combinatorial/double_factorial.html) : computes the factorial of unsigned integral values [`eve::factorial`](combinatorial/factorial.html) : computes the factorial of unsigned integral values [`eve::fibonacci`](combinatorial/fibonacci.html) : computes the nth term of a fibonacci sequence [`eve::gcd`](combinatorial/gcd.html) : computes the greater common divisor [`eve::lcm`](combinatorial/lcm.html) : computes the least common multiple [`eve::nth_prime`](combinatorial/nth_prime.html) : computes the nth prime number for n less than 10000 [`eve::prime_floor`](combinatorial/prime_floor.html) : computes the largest prime number less or equal to n for n less or equal to 104729 [`eve::prime_ceil`](combinatorial/prime_ceil.html) : computes the smallest prime number greater or equal to n for n less or equal to 104729 ----------------------------------------------------------------------------------------------------
