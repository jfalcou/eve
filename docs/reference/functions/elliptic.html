**Expressive Vector Engine** (insert ../../crumbs.html here) # Special functions These functions provide scalar and SIMD version of some special functions in namespace `eve`. The naming of these functions tries to follow the standard. - if the standard library implements the function, its name will be identical to the standard name - else if the boost math libary implements the function, its name will be identical to the library name - else if Octave implements the function, its name will be identical to the octave name ## Functions **Convenience header:** [`eve::am`](elliptic/am.html) : computes the Jacobi's elliptic amplitude function [`eve::ellint_rc`](elliptic/ellint_rc.html) : computes the Carlson's degenerate elliptic integral $\displaystyle R_C(x, y) = \frac12 \int_{0}^{\infty} (t+x)^{-1/2}(t+y)^{-1} \mbox{d}t$ [`eve::ellint_rd`](elliptic/ellint_rd.html) : computes the Carlson's elliptic integral $\displaystyle R_D(x, y, z) = \frac32 \int_{0}^{\infty} [(t+x)(t+y)]^{-1/2} (t+z)^{-3/2} \mbox{d}t$ [`eve::ellint_rf`](elliptic/ellint_rf.html) : computes the Carlson's elliptic integral $\displaystyle R_F(x, y) = \frac12 \int_{0}^{\infty} [(t+x)(t+y)]^{-1/2} (t+z)^{-3/2} \mbox{d}t$ [`eve::ellint_rj`](elliptic/ellint_rj.html) : computes the Carlson's elliptic integral $\displaystyle R_J(x, y, z, p) = \frac32 \int_{0}^{\infty} (t+p)^{-1}[(t+x)(t+y)(t+z)]^{-1/2}\mbox{d}t$ [`eve::ellint_rg`](elliptic/ellint_rg.html) : computes the Carlson's elliptic integral $\displaystyle R_G(x, y, z) = \frac1{4\pi} \int_{0}^{2\pi}\int_{0}^{\pi} \sqrt{x\sin^2\theta\cos^2\phi +y\sin^2\theta\sin^2\phi +z\cos^2\theta} \mbox{d}\theta\mbox{d}\phi$ [`eve::ellint_d`](elliptic/ellint_d.html) : computes the complete and incomplete D elliptic integrals : $\displaystyle D(\phi,k) = \int_0^{\phi} \frac{\sin^2 t}{\sqrt{1-k^2\sin^2 t}}\mbox{d}t$ [`eve::ellint_1`](elliptic/ellint_1.html) : computes the complete and incomplete elliptic integrals of the first kind : $\displaystyle F(\phi,k) = \int_0^{\phi} \frac{\mbox{d}t}{\sqrt{1-k^2\sin^2 t}}$ [`eve::ellint_2`](elliptic/ellint_2.html) : computes the complete and incomplete elliptic integrals of the second kind : $\displaystyle E(\phi,k) = \int_0^{\phi} \sqrt{1-k^2\sin^2 t} \mbox{d}t$ [`eve::ellint_3`](elliptic/ellint_3.html) : computes the complete and incomplete elliptic integrals of the third kind : $\displaystyle E(n,\phi,k) = \int_0^{\phi} \frac{\mbox{d}t}{(1-n\sin^2 t)\sqrt{1-k^2\sin^2 t}} \mbox{d}t$ [`eve::jacobi_zeta`](elliptic/jacobi_zeta.html) : computes the jacobi_zeta elliptic integral [`eve::heuman_lambda`](elliptic/heuman_lambda.html) : computes the heuman_lambda elliptic integral [`eve::ellpj`](elliptic/ellipj.html) : computes simultaneously the values of the jacobi elliptic functions sn, cn and dn [`eve::ellpk`](elliptic/ellipk.html) : computes simultaneously the values of the jacobi elliptic functions sn, cn and dn [`eve::ellpke`](elliptic/ellipke.html) : computes simultaneously the values of the complete elliptic integral of the first and second kind
