**Expressive Vector Engine** (insert ../../crumbs.html here) # Special functions These functions provide scalar and SIMD version of some special functions in namespace `eve`. The naming of these functions tries to follow the standard. - if the standard library implements the function, its name will be identical to the standard name - else if the boost math libary implements the function, its name will be identical to the library name - else if Octave implements the function, its name will be identical to the octave name ## Functions **Convenience header:** [`eve::am`](elliptic/am.html) : computes the Jacobi's elliptic amplitude function [`eve::ellint_rf`](elliptic/ellint_rf.html) : computes the Carlson's elliptic integral of the first kind $\displaystyle R_F(x, y, z) = \frac12 \int_{0}^{\infty} [(t+x)(t+y)(t+z)]^{-1/2} \mbox{d}t$ [`eve::ellint_rd`](elliptic/ellint_rd.html) : computes the Carlson's elliptic integral of the second kind $\displaystyle R_D(x, y, z) = \frac32 \int_{0}^{\infty} [(t+x)(t+y)]^{-1/2} (t+z)^{-3/2} \mbox{d}t$ [`eve::ellie`](elliptic/ellie.html) : computes the incomplete elliptic integral of the first kind : $\displaystyle E(a_0, a_1) = \int_0^{a_0} \sqrt{1-a_1\sin^2 t} \mbox{d}t$ [`eve::ellik`](elliptic/ellik.html) : computes the incomplete elliptic integral of the second kind: $\displaystyle F(a_0, a_1) = \int_0^{a_0} \frac1{\sqrt{1-a_1\sin^2 t}} \mbox{d}t$ [`eve::ellint_1`](elliptic/ellint_1.html) : computes the complete elliptic integral of the first kind : $\displaystyle K(a_0) = \int_0^{\pi/2} \sqrt{1-a_0^2\sin^2 t} \mbox{d}t$ [`eve::ellint_2`](elliptic/ellint_2.html) : computes the complete elliptic integral of the second kind : $\displaystyle E(a_0) = \int_0^{\pi/2} \sqrt{1-a_0^2\sin^2 t} \mbox{d}t$ [`eve::ellpj`](elliptic/ellipj.html) : computes simultaneously the values of the jacobi elliptic functions sn, cn and dn [`eve::ellpk`](elliptic/ellipk.html) : computes simultaneously the values of the jacobi elliptic functions sn, cn and dn [`eve::ellpke`](elliptic/ellipke.html) : computes simultaneously the values of the complete elliptic integral of the first and second kind ----------------------------------------------------------------------------------------------------
