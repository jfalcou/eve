**Expressive Vector Engine** `eve::is_negative`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the is_negative logical value. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise_) is_negative operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- For signed types The call `is_negative(x)` [element-wise](../../../glossary.html#value) returns true if and only if the bit of sign (most significant bit) is set. !!! WARNING this function coincides with `is_ltz` on [integral real values](../../concepts.html#value), but for [floating real values](../../concepts.html#value) `T`, `is_negative(Mzero<`T`>)` is true and if `n` is a Nan the result depends of the bit of sign of `n` which can be out of control although not undefined.
