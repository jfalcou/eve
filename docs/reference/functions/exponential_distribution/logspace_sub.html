**Expressive Vector Engine** `eve::logspace_sub`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the logspace_sub operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) logarithm of a difference from the logarithms of the terms. Parameters ---------------------------------------------------------------------------------------------------- `arg0`, `arg1`, `args`, ... : [values](../../concepts.html#value). Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff_nth< N >(logspace_sub)(arg0,arg1,args...)` computes the partial diff of the function relative to its Nth parameter. The returned value is 0 if N is greater that the actual number of parameters, otherwise if the parameters are $x_1, ... x_n$ returns $\displaystyle\frac{s e^{x_N}}{e^{x_1} - ... - e^{x_n}}$ where $s$ is $1$ if $N = 1$ and $-1$ otherwise. Return value ---------------------------------------------------------------------------------------------------- Similar to `log(exp(arg0) - (exp(arg1) + exp(args)...))`;.
