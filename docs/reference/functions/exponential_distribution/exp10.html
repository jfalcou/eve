**Expressive Vector Engine** `eve::exp10`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of exp10. Member functions ==================================================================================================== Performs $10^x$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) exponential of base 10 of the input. In partucular, for floating inputs: * If the element is $\pm0$, $1$ is returned * If the element is $-\infty$, $+0$ is returned * If the element is $\infty$, $\infty$ is returned * If the element is a `NaN`, `NaN` is returned Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(exp10)(x)` computes the derivative of the function at `x`.
