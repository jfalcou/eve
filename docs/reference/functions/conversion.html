**Expressive Vector Engine** (insert ../../crumbs.html here) # Conversion functions These functions provide scalar and SIMD version of type conversions including specific, optimized float <==> int conversions and a general type casting facility. All these functions operates on the basis that a type conversion does not modify a [value](../concepts.html#value)'s cardinal, i.e a [scalar](../concepts.html#scalar_value) is converted to a [scalar](../concepts.html#scalar_value) and a [SIMD value](../concepts.html#simd_value) is converted to a [SIMD value](../concepts.html#simd_value) with the same cardinal. ## Functions **Convenience header:** [`eve::convert`](conversion/convert.html) : converts a value to another type [`eve::tofloat`](conversion/tofloat.html) : converts a value to the corresponding floating point type [`eve::toint`](conversion/toint.html) : converts a value to the corresponding signed integer [`eve::touint`](conversion/touint.html) : converts a value to the corresponding unsigned integer ----------------------------------------------------------------------------------------------------
