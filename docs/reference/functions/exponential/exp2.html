(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of exp2. Member functions ==================================================================================================== Performs $2^x$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) exponential of base 2 of the input. In partucular, for floating inputs: * If the element is $\pm0$, $1$ is returned * If the element is $-\infty$, $+0$ is returned * If the element is $\infty$, $\infty$ is returned * If the element is a `NaN`, `NaN` is returned (insert ../../src/exp2.src.html here) (insert ../../out/exp2.out.html here) See Also ==================================================================================================== [exp](exp.html), [exp10](exp10.html), [expm1](expm1.html).
