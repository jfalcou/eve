**Expressive Vector Engine** `eve::log10`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of the base 10 logarithm. Member functions ==================================================================================================== Performs $\log x/ \log 10$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) the base 10 logarithm of `x` In partucular, for floating inputs: * If the element is $\pm0$, $-\infty$ is returned. * If the element is $1$, $+0$ is returned. * If the element is $\infty$, $\infty$ is returned. * If the element is less than 0, `NaN` is returned.
