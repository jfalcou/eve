**Expressive Vector Engine** `eve::expm1`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of expm1. Member functions ==================================================================================================== Performs $e^x-1$. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) exponential input minus one, with good accuracy, even for small values of `x` In partucular, for floating inputs: * If the element is $\pm0$, $\pm0$ is returned * If the element is $-\infty$, $-1$ is returned * If the element is $\infty$, $\infty$ is returned * If the element is a `NaN`, `NaN` is returned Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(expm1)(x)` computes the derivative of the function at `x`.
