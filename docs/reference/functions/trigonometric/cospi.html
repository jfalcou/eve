**Expressive Vector Engine** `eve::cospi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cospi. Member functions ==================================================================================================== Performs cosine computation with argument in $\pi$ multiples. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) cosine of the input expressed in $\pi$ multiples. The call `cospi(x)` is semantically equivalent to $cos(\pi x)$. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== As all direct trigonometric functions, `cospi` supports the restricted, small, medium and big decorators.
