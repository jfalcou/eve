**Expressive Vector Engine** `eve::cos`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cos. Member functions ==================================================================================================== Performs cosine computation. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) cosine of the input. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `Nan`, `Nan` is returned. Supported decorators ==================================================================================================== As all direct trigonometric functions, `cos` supports the restricted, small, medium and big decorators. Click to see the [properties and rationale](../trigonometric.html) [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(cos)(x)` computes the derivative of the function at `x`.
