**Expressive Vector Engine** `eve::cot`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cot. Member functions ==================================================================================================== Performs cotangent computation. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) cotangent of the input. In particular: * If the element is $\pm0$, $\pm\infty$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `NaN`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `cot` supports the restricted, small, medium and big decorators.
