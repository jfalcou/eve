**Expressive Vector Engine** `eve::cscpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cscpi. Member functions ==================================================================================================== Performs cosecant computation with argument in $\pi$ multiples. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) cotine of the input expressed in $\pi$ multiples. The call `cscpi(x)` is semantically equivalent to $cot(\pi x)$. In particular: * If the element is $\pm0$, $\pm\infty$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `cscpi` supports the restricted, small, medium and big decorators.
