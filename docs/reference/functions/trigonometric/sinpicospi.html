**Expressive Vector Engine** `eve::sinpicospi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the simultaneous computation of sin an cos of an argument in . Member functions ==================================================================================================== Performs sine and cosine computations in $\pi$ multiples. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- The computation returns a pair and is semantically equivalent to `{sinpi(x), cospi(x)}` Supported Decorators ==================================================================================================== As all direct trigonometric functions, `sinpicospi` supports the restricted_, small_, medium_ and big_ decorators.
