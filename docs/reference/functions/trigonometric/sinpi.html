**Expressive Vector Engine** `eve::sinpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sinpi. Member functions ==================================================================================================== Performs sine computation with argument in $\pi$ multiples. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) sinine of the input expressed in $\pi$ multiples. The call `sinpi(x)` is semantically equivalent to $sin(\pi x)$. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `sinpi` supports the restricted, small, medium and big decorators.
