**Expressive Vector Engine** `eve::tan`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of tan. Member functions ==================================================================================================== Performs tangent computation. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) tangent of the input. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== As all direct trigonometric functions, `tan` supports the restricted, small, medium and big decorators. Click to see the [properties and rationale](../trigonometric.html) [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(tan)(x)` computes the derivative of the function at `x`.
