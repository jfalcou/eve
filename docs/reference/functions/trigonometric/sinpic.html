**Expressive Vector Engine** `eve::sinpic`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sinpic. Member functions ==================================================================================================== Performs pi-sine cardinal computation. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) sine of the input times $\pi$ divided by the input times $\pi$. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, 0 is returned. * If the element is a `Nan`, `NaN` is returned.
