**Expressive Vector Engine** `eve::sec`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sec. Member functions ==================================================================================================== Performs secant computation. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) secant of the input. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `NaN`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `sec` supports the restricted, small, medium and big decorators.
