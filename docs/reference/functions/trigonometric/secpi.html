**Expressive Vector Engine** `eve::secpi`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of secpi. Member functions ==================================================================================================== Performs secant computation with argument in $\pi$ multiples. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) secine of the input expressed in $\pi$ multiples. The call `secpi(x)` is semantically equivalent to $sec(\pi x)$. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `Nan`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `secpi` supports the restricted_, small_, medium_ and big_ decorators.
