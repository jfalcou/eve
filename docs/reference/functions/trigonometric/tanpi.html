(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of tanpi. Member functions ==================================================================================================== Performs tanpigent computation with argument in $\pi$ multiples.. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) tangent of the input expressed in $\pi$ multiples. The call `tanpi(x)` is semantically equivalent to tan(\pi x)$. In particular: * If the element is $\pm0$, $\pm0$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== As all direct trigonometric functions, `tanpi` supports the restricted, small, medium and big decorators.
