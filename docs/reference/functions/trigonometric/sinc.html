**Expressive Vector Engine** `eve::sinc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sinc. Member functions ==================================================================================================== Performs sine cardinal computation. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) sine of the input divided by the input. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, 0 is returned. * If the element is a `Nan`, `NaN` is returned. Supported decorators ==================================================================================================== [`diff`](../decorators.html#diff) : **Required header: ** : The expression `diff(sinc)(x)` computes the derivative of the function at `x`.
