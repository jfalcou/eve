**Expressive Vector Engine** `eve::cotd`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cotd. Member functions ==================================================================================================== Performs cotangent computation with argument in degrees. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) cotangent of the input expressed in degrees. In particular: * If the element is $\pm0$, $\pm\infty$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== As all direct trigonometric functions, `cotd` supports the restricted, small, medium and big decorators.
