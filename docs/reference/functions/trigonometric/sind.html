**Expressive Vector Engine** `eve::sind`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of sind. Member functions ==================================================================================================== Performs sine computation of argument in degrees. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) sine of the input expressed in degrees. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `NaN`, `NaN` is returned. Supported decorators ==================================================================================================== As all direct trigonometric functions, `sind` supports the restricted, small, medium and big decorators.
