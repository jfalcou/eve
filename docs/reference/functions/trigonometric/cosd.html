**Expressive Vector Engine** `eve::cosd`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the computation of cosd. Member functions ==================================================================================================== Performs cosine computation of argument in degrees. Parameters ---------------------------------------------------------------------------------------------------- `x` : [floating real value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Returns the [element-wise](../../../glossary.html#elment-wise) cosine of the input expressed in degrees. In particular: * If the element is $\pm0$, $1$ is returned. * If the element is $\pm\infty$, Nan is returned. * If the element is a `NaN`, `NaN` is returned. Supported Decorators ==================================================================================================== As all direct trigonometric functions, `cosd` supports the restricted, small, medium and big decorators.
