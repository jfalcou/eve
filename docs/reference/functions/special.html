**Expressive Vector Engine** (insert ../../crumbs.html here) # Special functions These functions provide scalar and SIMD version of some special functions in namespace `eve`. The naming of these functions tries to follow the standard. - if the standard library implements the function, its name will be identical to the standard name - else if the boost math libary implements the function, its name will be identical to the library name - else if Octave implements the function, its name will be identical to the octave name ## Functions **Convenience header:** [`eve::beta`](special/beta.html) : computes $\displaystyle \mbox{B}(x,y)=\int_0^1 t^{x-1}(1-t)^{y-1}\mbox{d}t$ [`eve::betainc`](special/betainc.html) : computes the lower normalized beta function: $\displaystyle \mbox{B}(s,x,y)=\frac{1}{\mbox{B}(x,y)}\int_0^s t^{x-1}(1-t)^{y-1}\mbox{d}t$ [`eve::betainc_inv`](special/betainc.html) : computes the inverse of the incomplete beta function relative to the first parameter. [`eve::dawson`](special/dawson.html) : computes $\displaystyle D_+(x)=e^{-x^2}\int_0^{x} e^{t^2} \mbox{d}t$ [`eve::digamma`](special/digamma.html) : computes the logarithmic diff of the $\Gamma$ function: $\displaystyle \frac{\Gamma^\prime(x)}{\Gamma(x)}$ [`eve::erf`](special/erf.html) : computes $\displaystyle \mbox{erf}(x)=\frac{2}{\sqrt\pi}\int_0^{x} e^{-t^2}\mbox{d}t$ [`eve::erfc`](special/erfc.html) : computes the complementary error function $1-\mbox{erf}(x)$ [`eve::erfcx`](special/erfcx.html) : computes the normalized complementary error function $\mbox{erfcx}(x) = e^{x^2} \mbox{erfc}(x)$ [`eve::erf_inv`](special/erf_inv.html) : computes the inverse of the $\mbox{erf}$ function [`eve::lbeta`](special/lbeta.html) : computes the logarithm of the absolute value of the beta function: $\log(\mbox{B}(x))$ [`eve::lgamma`](special/lgamma.html) : computes the logarithm of the absolute value of the $\Gamma$ function: $\log(|\Gamma(x)|)$ [`eve::exp_int`](special/exp_int.html) : computes the exponential integral $\displaystyle \mbox{E}_{n}(x)=\int_0^\infty \frac{e^{-xt}}{t^{n}} \mbox{d}t$ [`eve::tgamma`](special/tgamma.html) : computes the (true) $\Gamma$ function: $\displaystyle \Gamma(x)=\int_0^\infty t^{x-1}e^{-t}\mbox{d}t$ [`eve::gamma_p`](special/gamma_p.html) : computes the normalized lower incomplete $\Gamma$ function: $\displaystyle \mbox{P}(x,y) = \frac{1}{\Gamma(x)}\int_0^{y} t^{x-1}e^{-t}\mbox{d}t$ [`eve::signgam`](special/signgam.html) : computes the sign of $\Gamma(x)$ [`eve::stirling`](special/stirling.html) : computes an approximation of the $\Gamma$ function by $\displaystyle \Gamma(x) \approx \sqrt{2 \pi} x^{x-\frac12} e^{-x} ( 1 + \frac1{x} P(\frac1{x}))$, where $P$ is a polynomial. [`eve::zeta`](special/zeta.html) : computes the Riemann $\zeta$ function: $\displaystyle \zeta(s)=\sum_{n=0}^\infty \frac1{n^s}$ ----------------------------------------------------------------------------------------------------
