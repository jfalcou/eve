**Expressive Vector Engine** `eve::average`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the mid-point operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) arithmetic average of the two [values](../../concepts.html#value). Parameters ---------------------------------------------------------------------------------------------------- `x`, `y` : [values](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Half the sum of `x` and `y`. No overflow occurs. The result type is the [compatibility result](../../concept.html#compatibility) of the two parameters. !!! WARNING If `x` and `y` are [real integral values](../../concepts.html#value) and the sum is odd, the result is a rounded value at a distance guaranteed to be less than or equal to 0.5 of the average floating value, but may differ by unity from the truncation given by `(x+y)/2`. Moreover, as some architectures provide simd intrinsics to perform the operation, the scalar results may differ by one unit from simd ones and is system dependent.
