Member functions ==================================================================================================== compute the [element-wise](../../../glossary.html#regular) smallest prime number greater or equal to n (if n is less than 104730). Parameters ---------------------------------------------------------------------------------------------------- `n` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- The result element type is the same as the input one unless a converter is applied.
