**Expressive Vector Engine** `eve::primes`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing unsigned integral primes. Member functions ==================================================================================================== compute the [element-wise](../../../glossary.html#regular) nth prime number (if n is less than 10000). Parameters ---------------------------------------------------------------------------------------------------- `n` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- The result element type is the same as the input one unless a converter is applied to `primes`. See the example below. Almost no computations are made, the results are from a lookup table. 2 is the first prime number (n=1) and `primes(0)` returns `0` which is used as incorrect computation. Note that is the result has been converted to a floating value incorrect compution will return nan !!! WARNING this function will return 0 (or nan) as soon as the correct result will not be exactly represented in the output type or (for the current implementation) if n is greater than 10000.
