**Expressive Vector Engine** `eve::lrising_factorial`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing lrising_factorial i.e. $\log\left(\frac{\Gamma(x+a)}{\Gamma(x)}\right)$ Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) lrising_factorial of the [value](../../concepts.html#value). `a` and `a+x` must be positive or the result is a Nan (see above for pedantic extension). Parameters ---------------------------------------------------------------------------------------------------- `a` : [value](../../concepts.html#value). `x` : [floating value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Supported decorators ==================================================================================================== [`raw`](../decorators.html#saturated) : **Required header: ** : The expression `raw(lrising_factorial)(a,x)` uses the crude formula with all its limitations and inacurracies and return a Nan if `a` and `a+x` are not both positive [`pedantic`](../decorators.html#saturated) : **Required header: ** : The expression `pedantic(lrising_factorial)(a,x)` uses reflection tricks and computes the function for all real `a` and `x`, and in fact computes the logarithm of the absolute value of the Pochammer symbol $\log\left|\frac{\Gamma(x+a)}{\Gamma(x)}\right|$ returning nan if the result in really undefined. [`diff`](../decorators.html#saturated) : **Required header: ** : The expression `diff_1st(lrising_factorial)(a,x)` and `diff_2nd(lrising_factorial)(a,x)` computes the derivative of the function relative to the first or second parameter respectively. These decorators can be combined to the other available ones : for instance the call `pedantic(diff_1st)(lrising_factorial)(a,x)` will compute the derivative of `pedantic(lrising_factorial)` relative to the first parameter.
