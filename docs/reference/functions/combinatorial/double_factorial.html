**Expressive Vector Engine** `eve::double_factorial`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing unsigned integral double_factorial. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) double_factorial of the [value](../../concepts.html#value). Parameters ---------------------------------------------------------------------------------------------------- `n` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- double factorial is defined by $\displaystyle (2n)!! = \prod_{i=1}^n (2i)$ and $\displaystyle (2n+1)!! = \prod_{i=0}^n (2i+1)$ The result element type is always double to try to avoid overflow and its cardinal is the same as the entry in case of an simd call. !!!
