(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Object representing the Lognormal distribution. Constructor ==================================================================================================== construct the lognormal_distribution object Parameters ---------------------------------------------------------------------------------------------------- `m` : [floating real value](../../concepts.html#value) or callable_zero object `s` : [floating real value](../../concepts.html#value) or callable_one object Optimizations ==================================================================================================== If s is one orm zero the second and third constructors can be used with the parameters eve::one or eve::zero, providing some optimization in further computations of the attributes. Also the constexpr template variable lognormal_distribution_01< T > is provided to optimize the standard case (m 0, s 1). In this case, T must be provided to allow choosing the output type of the constant attributes. Supported attributes ==================================================================================================== - functional attributes - cdf (raw(cdf) provides for float based entries a less accurate but faster computation) - pdf - invcdf - constant attributes - median - mean - mode - skewness - kurtosis - stdev - var - entropy See the example below.
