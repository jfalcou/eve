**Expressive Vector Engine** `eve::normal`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Object representing the Normal distribution. Constructor ==================================================================================================== construct the normal object Parameters ---------------------------------------------------------------------------------------------------- `m` : [floating real value](../../concepts.html#value) or callable_zero object (mean of the distribution) `s` : [floating real value](../../concepts.html#value) or callable_one object (standard deviation (>0) of the distribution) Optimizations ==================================================================================================== If the standard deviation is one or the mean zero the second and third constructors can be used with the parameters eve::one or eve::zero, providing some optimization in further computations of the attributes. Also the constexpr template variable normal_01< T > is provided to optimize the standard reduced case (mean 0, stdev 1). In this case, T must be provided to allow choosing the output type of the constant attributes. Supported attributes ==================================================================================================== - functional attributes - cdf (raw(cdf) provides for float based entries a less accurate but faster computation) - pdf - invcdf - mgf - constant attributes - median - mean - mode - skewness - kurtosis - stdev - var - entropy See the example below.
