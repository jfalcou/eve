**Expressive Vector Engine** `eve::cauchy`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Object representing the Cauchy distribution. Constructor ==================================================================================================== construct the cauchy object Parameters ---------------------------------------------------------------------------------------------------- `m` : [floating real value](../../concepts.html#value) or callable_zero object (median of the distribution) `s` : [floating real value](../../concepts.html#value) or callable_one object (scale (>0) of the distribution) Optimizations ==================================================================================================== If the scale is one or the median zero the second and third constructors can be used with the parameters eve::one or eve::zero, providing some optimization in further computations of the attributes. Also the constexpr template variable cauchy_01< T > is provided to optimize the standard reduced case (median 0, scale 1). In this case, T must be provided to allow choosing the output type of the constant attributes. Supported attributes ==================================================================================================== - functional attributes - cdf - pdf - invcdf - constant attributes - median - scale - mode - entropy - fisher cauchy distribution have no mean nor variance or standard deviation.
