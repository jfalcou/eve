Constructor ==================================================================================================== construct the uniform_discrete_distribution object Parameters ---------------------------------------------------------------------------------------------------- `a` : [floating real value](../../concepts.html#value) lower bound, must be a flint `b` : [floating real value](../../concepts.html#value) upper bound, must be a flint strictly greater than a ` Optimizations ==================================================================================================== Also the constexpr template variable uniform_discrete_distribution_01< T > is provided to optimize the case of interval of the (0,1) sequence (i.e. fair dice tossing). In this case, T must be provided to allow choosing the output type of the constant attributes. Supported attributes ==================================================================================================== - functional attributes - cdf - pdf - mgf - invcdf - constant attributes - median - mean - mode - skewness - kurtosis - stdev - var - pseudo random generator if the distribution parameters are scalar.
