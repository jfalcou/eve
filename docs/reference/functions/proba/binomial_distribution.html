(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Object representing the Binomial distribution. Constructor ==================================================================================================== construct the binomial_distribution object Parameters ---------------------------------------------------------------------------------------------------- `n` : [floating real value](../../concepts.html#value) (integral or flint positive value) or callable_one object (number of experiments) `p` : [floating real value](../../concepts.html#value) (must be between 0 and 1) or callable_half object (probability of success) Optimizations ==================================================================================================== If the number of experiments is one or the probabilty of success $\frac12$, the third and fourth constructors can be used with the parameters eve::one or eve::half, providing some optimization in further computations of the attributes. Also the constexpr template variable bernouilli< T > is provided to optimize the case of fair coin tossing (n = 1, p = 0.5). In this case, T must be provided to allow choosing the output type of the constant attributes. Supported attributes ==================================================================================================== - functional attributes - cdf - pmf - mgf - constant attributes - median (approximate) - mean - mode - skewness - kurtosis - stdev - var See the example below.
