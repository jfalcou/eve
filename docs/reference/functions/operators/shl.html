**Expressive Vector Engine** `eve::shl`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the arithmetic left shift operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#regular) arithmetic left shift of the first parameter by the second one. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). `n` : [integral value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes the [element-wise](../../../glossary.html#element-wise) arithmetic left shift of the first parameter by the second one. the call `shl(x, n)` is equivalent to `x << n` if `x` is an [simd value](../../concepts.html#value). The types must share the same cardinal or be scalar and if $N$ is the size in bits of the element type of `T`, all [elements](../../../glossary.html#element) of n must belong to the interval: `[0, N[` or the result is undefined. !!! WARNING Although the infix notation with `<<` is supported, the `<<` operator on standard scalar types is the original one and so can not be overloaded on standard floating parameters due to **C++** limitations.
