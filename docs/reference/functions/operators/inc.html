**Expressive Vector Engine** `eve::inc`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the inc unary operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) increment by one operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) a value with the same type as `x`. The result is `x+1`. Supported Incorators ==================================================================================================== [`saturated_`](../incorators.html#pedantic_) : The call `saturated_(inc)(x)` computes the saturated opposite of `x`. The only interest of this behaviour is that for integral type T `saturated_(inc)(Valmax< T >())` returns `Valmax< T >()`. Masked calls ==================================================================================================== The call `inc[cond](x)` (resp. `saturated_(inc[cond])(x)`) computes the opposite (resp the saturated opposite of `x`) if the condition `cond` is satisfied else returns `x`.
