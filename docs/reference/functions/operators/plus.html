**Expressive Vector Engine** `eve::plus`
(insert ../../../crumbs.html here) Synopsis ==================================================================================================== **Required header:** Function object representing the plus unary operation. Member functions ==================================================================================================== Performs the [element-wise](../../../glossary.html#element-wise) plus operation on its parameter. Parameters ---------------------------------------------------------------------------------------------------- `x` : [value](../../concepts.html#value). Return value ---------------------------------------------------------------------------------------------------- Computes [element-wise](../../../glossary.html#elment-wise) `+x` which generally is a no-op. !!! WARNING Although the operator notation with `+` is supported, the `+` operator on standard scalar type is the original one and so can lead to automatic C++ promotion. Masked calls ==================================================================================================== The [masked](../../../tutorial/masking.html) calls `plus[cond](x)` and `saturated(plus[cond])(x)` where `cond` satisfies the [conditionnal-expression](../../concepts.html#conditionnal) requirements are supported. For this peculiar operation, as `plus` acts merely as identity, the masked calls are really interesting only if the `cond` clause invloves an `else_` part (see [the masking tutorial](../../../tutorial/masking.html)).
