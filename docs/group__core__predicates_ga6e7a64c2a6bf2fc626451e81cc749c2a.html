Semantic Modifiers

  • Masked Call

    The call eve;is_denormal[mask](x) provides a masked version of eve::is_denormal which is equivalent to if_else (mask, is_denormal(x), eve::false( eve::as(x))).

    Example

    #include <eve/module/core.hpp>
    #include <eve/wide.hpp>

    wide_ft pf = {0.0f,1.0f, -1.0f, -2.0f,
    eve::mindenormal(eve::as<float>()), -eve::mindenormal(eve::as<float>()),
    eve::minf(eve::as<float>()), eve::nan(eve::as<float>())};
