Detailed Description

template<typename Type, std::size_t Alignment = sizeof(Type) * expected_cardinal_v<Type>>
struct eve::aligned_ptr< Type, Alignment >

Wrapper for non-owning aligned pointers.

Required header: #include <eve/memory/aligned_ptr.hpp>

aligned_ptr is a non-owning pointer that binds a preexisting, properly aligned memory and prevent any alignment-altering operations to be performed. Rebinding to another pointer is allowed only if the new pointer's alignment is compatible with the current aligned_ptr constraints.

An aligned_ptr may alternatively own no object, in which case it is called empty.

Template Parameters
TypePointee type
AlignmentAlignment constraint to uphold. Code is ill-formed if it isn't a power of 2. Default value is computed to be compatible with current SIMD ABI requirements.
