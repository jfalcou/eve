Detailed Description

These functions provide scalar and SIMD version of basic bitwise functions.

To keep the results homogeneous and coherent between SIMD and scalar mode, the types usable in those functions are generally required to satisfy eve::bit_compatible_values

Convenience header:

#include <eve/function/bits.hpp>
