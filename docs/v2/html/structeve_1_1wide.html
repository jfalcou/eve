Detailed Description

template<typename Type, typename Size, typename ABI>
struct eve::wide< Type, Size, ABI >

Wrapper for SIMD registers.

wide is an architecture-agnostic representation of a low-level SIMD register and provides standardized API to access informations, compute values and manipulate such register.

Template Parameters
TypeType of value to store in the register
SizeCardinal of the register. By default, the best cardinal for current architecture is selected.

Inherits eve::detail::wide_cardinal< Size >, eve::detail::wide_ops< Derived >, and eve::detail::wide_storage< as_register_t< Type, Size, ABI > >.
